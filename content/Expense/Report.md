# Report — Expense

**Report** ในส่วน Expense คือรายงานสรุปค่าใช้จ่ายทั้งหมดตามช่วงเวลาที่เลือก

---

## การดูรายงาน

1. เลือกช่วงวันที่จาก **Date Filter**
2. เลือกกรองตาม Trip หรือ Item Type (ถ้าต้องการ)
3. ระบบสร้างรายงานสรุปอัตโนมัติ
4. กดปุ่ม **Export** เพื่อดาวน์โหลดเป็นไฟล์ Excel

---

## ข้อมูลในรายงาน

- สรุปค่าใช้จ่ายรายเดือน/รายปี
- เปรียบเทียบ Advance vs Actual Expense
- สรุปยอดคงเหลือตาม Trip
- รายละเอียดค่าใช้จ่ายแยกตาม Item Type
