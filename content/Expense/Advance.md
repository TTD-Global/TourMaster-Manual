# Advance — เบิกล่วงหน้า

**Advance** คือการบันทึกและจัดการค่าใช้จ่ายที่เบิกล่วงหน้าสำหรับ Trip หรือโปรแกรมทัวร์ต่างๆ ก่อนที่จะออกเดินทาง

---

## ภาพรวม

หน้า Advance แสดงรายการเบิกล่วงหน้าทั้งหมด ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Trip / Program | ชื่อ Trip หรือโปรแกรม |
| Travel Date | วันที่เดินทาง |
| Amount | จำนวนเงินที่เบิก |
| Status | สถานะ (Pending / Approved / Rejected) |
| Action | ปุ่มจัดการ |

---

## การสร้างรายการ Advance

1. กดปุ่ม **Create** หรือ **+ Add**
2. เลือก **Trip / Program** ที่ต้องการเบิก
3. กรอก **Amount** จำนวนเงินที่ต้องการเบิก
4. เลือก **Item Type** ประเภทค่าใช้จ่าย
5. กรอกรายละเอียดและ Remark (ถ้ามี)
6. กดบันทึก

---

## สถานะรายการ Advance

| สถานะ | ความหมาย |
|-------|----------|
| Pending | รออนุมัติ |
| Approved | อนุมัติแล้ว |
| Rejected | ปฏิเสธ |
| Settled | ชำระเงินแล้ว |

:::info
รายการ Advance ที่ได้รับการอนุมัติแล้ว จะถูกนำไปเปรียบเทียบกับค่าใช้จ่ายจริง (Expenses) ในภายหลัง
:::
