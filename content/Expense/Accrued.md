# Balance — สรุปยอดคงเหลือ

**Balance** คือหน้าสรุปยอดเงินคงเหลือของแต่ละ Trip เปรียบเทียบระหว่างเงิน Advance ที่เบิกไปกับค่าใช้จ่ายจริงที่เกิดขึ้น

---

## ภาพรวม

หน้า Balance แสดงสรุปยอดคงเหลือต่อ Trip

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| Trip / Program | ชื่อ Trip หรือโปรแกรม |
| Travel Date | วันที่เดินทาง |
| Total Advance | ยอดเงิน Advance รวม |
| Total Expenses | ยอดค่าใช้จ่ายจริงรวม |
| Balance | ยอดคงเหลือ (Advance - Expenses) |
| Status | สถานะ (คืนเงิน / เบิกเพิ่ม / ตรงกัน) |

---

## การตีความ Balance

- **Balance > 0** (บวก) — มีเงินเหลือต้องคืนกองกลาง
- **Balance = 0** — ค่าใช้จ่ายตรงกับ Advance พอดี
- **Balance < 0** (ลบ) — ใช้เกิน Advance ต้องเบิกเพิ่ม

---

## การ Export รายงาน

กดปุ่ม **Export** เพื่อส่งออกรายงาน Balance ทั้งหมดเป็นไฟล์ Excel สำหรับใช้ในการบัญชี

:::info
Balance จะคำนวณอัตโนมัติเมื่อมีการบันทึกทั้ง Advance และ Expenses ครบถ้วน
:::
