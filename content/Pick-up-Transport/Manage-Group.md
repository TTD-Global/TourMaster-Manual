# Manage Group — Pick up Transport

**Manage Group** ในส่วน Pick up Transport คือการจัดกลุ่มลูกค้าที่ต้องการรับส่งจากจุด Pick up ต่างๆ ตามวันที่ เพื่อจัดสรรยานพาหนะได้อย่างมีประสิทธิภาพ

---

## ภาพรวม

ตารางแสดงรายการ Pick up Transport ที่จัดกลุ่มไว้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อโปรแกรม |
| Destination | จุดรับ (Pick up point) |
| Unit | จำนวน/หน่วย |
| Total | ยอดรวม |
| Action | ปุ่มจัดการ |

---

## การกรองข้อมูล

- **Date Filter** — เลือกวันที่ที่ต้องการดูรายการ Pick up
- **Search Program** — ค้นหาโปรแกรมตามชื่อ
