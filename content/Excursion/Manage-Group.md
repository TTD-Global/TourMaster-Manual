# Manage Group — Excursion

**Manage Group** ในส่วน Excursion คือการจัดกลุ่มโปรแกรม Excursion (ทริปเสริม/กิจกรรมเสริม) ตามวันที่ เพื่อให้สามารถจัดสรร Supplier หรือรถ Own Vehicle ได้อย่างมีประสิทธิภาพ

---

## ภาพรวม

ตารางแสดงรายการ Excursion ที่จัดกลุ่มไว้ ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อโปรแกรม Excursion |
| Destination | จุดหมายปลายทาง |
| Unit | จำนวน/หน่วย |
| Total | ยอดรวม |
| Action | ปุ่มจัดการ |

---

## การกรองข้อมูล

- **Date Filter** — เลือกวันที่ที่ต้องการดูรายการ Excursion
- **Search Program** — ค้นหาโปรแกรมตามชื่อ

---

## Action

คลิกปุ่ม **Action** ที่แถวรายการเพื่อ
- เข้าดูรายละเอียดการจัดกลุ่ม
- จัดสรร Supplier Group หรือ Own Vehicle Group ให้กับ Excursion นั้น

:::info
การ Manage Group จะแสดงเฉพาะรายการ Excursion ที่มี Booking ยืนยันแล้ว (Confirmed) ในวันที่ที่เลือก
:::
