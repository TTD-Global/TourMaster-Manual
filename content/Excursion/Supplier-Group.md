# Supplier Group — Excursion

**Supplier Group** ในส่วน Excursion คือการจัดสรร Supplier ภายนอก (รถหรือบริการที่จ้างจากบริษัทอื่น) ให้กับรายการ Excursion ในแต่ละวัน

---

## ภาพรวม

หน้านี้แสดงรายการ Excursion ที่ต้องใช้รถ/บริการจาก Supplier ภายนอก ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อโปรแกรม Excursion |
| Destination | จุดหมายปลายทาง |
| Unit | จำนวน/หน่วย |
| Total | ยอดรวม |
| Action | ปุ่มจัดการ Supplier |

---

## การจัดสรร Supplier

1. เลือกวันที่จาก **Date Filter**
2. ค้นหาโปรแกรมด้วย **Search Program** (ถ้าต้องการ)
3. คลิก **Action** ที่รายการที่ต้องการ
4. เลือก Supplier และยานพาหนะที่ต้องการจัดสรร
5. กดบันทึก

:::tip
ควรตั้งค่า Supplier ไว้ในเมนู Setting > Vehicle > Supplier List ก่อน เพื่อให้มีข้อมูลให้เลือก
:::
