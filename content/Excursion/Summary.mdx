# Summary

หน้า **Summary** แสดงภาพรวมรายการ Excursion ที่มี Booking Confirmed ในแต่ละวัน เพื่อให้เจ้าหน้าที่สามารถจัดสรรยานพาหนะได้จากหน้าเดียว

เข้าถึงได้จาก **Excursion → Summary**

---

## ตารางรายการ

![Excursion Summary](/img/Excursion/excursion-summary.png)

ตารางแสดงรายการ Program Excursion ที่มี Booking ในวันที่เลือก ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Program Name** | ชื่อโปรแกรม Excursion |
| **Destination** | จุดหมายปลายทาง |
| **Unit** | หน่วยและจำนวนผู้เดินทาง เช่น Person x 44 |
| **Total** | จำนวนผู้เดินทางรวม |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

แถวสุดท้ายแสดง **Total** รวมจำนวนผู้เดินทางทั้งหมดของวันนั้น

:::info
หน้านี้แสดงเฉพาะ Program ที่มี Booking สถานะ **Confirmed** ในวันที่เลือกเท่านั้น
:::

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Date Filter** | เลือกวันที่เพื่อดูรายการ Excursion |
| **Search Program** | ค้นหา Program ตามชื่อ |
| **Manage Group** | เปิดหน้าจัดการกลุ่มแบบรวม |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อเลือกวิธีจัดสรรยานพาหนะ

![Excursion Summary — Action](/img/Excursion/excursion-summary-action.png)

| ตัวเลือก | คำอธิบาย |
|---------|----------|
| **Supplier Group** | ไปยังหน้าจัดสรรยานพาหนะจาก Supplier ภายนอก |
| **Own Vehicle Group** | ไปยังหน้าจัดสรรยานพาหนะของบริษัทเอง |
