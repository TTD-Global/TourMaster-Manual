# Own Vehicle Group — Excursion

**Own Vehicle Group** ในส่วน Excursion คือการจัดสรรรถ/ยานพาหนะของบริษัทเอง (Own Vehicle) ให้กับรายการ Excursion ในแต่ละวัน

---

## ภาพรวม

หน้านี้แสดงรายการ Excursion ที่ต้องใช้รถของบริษัทเอง ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อโปรแกรม Excursion |
| Destination | จุดหมายปลายทาง |
| Unit | จำนวน/หน่วย |
| Total | ยอดรวม |
| Action | ปุ่มจัดสรรรถ Own Vehicle |

---

## การจัดสรร Own Vehicle

1. เลือกวันที่จาก **Date Filter**
2. คลิก **Action** ที่รายการที่ต้องการ
3. เลือกรถที่ต้องการจัดสรรจากรายการ Own Vehicle ที่มีอยู่
4. กดบันทึก

:::tip
ต้องเพิ่มรถ Own Vehicle ไว้ในเมนู Setting > Vehicle > Own Vehicle ก่อน เพื่อให้มีข้อมูลให้เลือก
:::
