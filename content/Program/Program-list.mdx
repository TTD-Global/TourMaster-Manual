**Program List** รายละเอียด Product หรือรายละเอียด Program ของ Tour นั้นๆ โดยมีรายละเอียดสำคัญดังนี้
- **ปุ่ม Create** สำหรับสร้าง Program ของ Tour 
- **Agent Quantity** แสดงจำนวน Agent ของ Program นั้นๆ
- **Status** แสดงสถานะ เปิด-ปิด การใช้งาน Program
- **Last Modify Date** แสดงวันที่ สร้างหรือแก้ไขล่าสุด
- **Action** สำหรับจัดการโปรแกรมทัวร์

---

## **Create**
เมื่อกด Create จะเข้าสู่หน้าสร้าง Program ของ Tour โดยมีข้อมูลที่ต้องกรอกทั้งหมด 5 ส่วนใหญ่ๆ ดังนี้

## **Program Detail** 
    หน้าต่างนี้กรอกข้อมูลเบื้องต้นของ Program
    - **Program Name** ชื่อของ Program นั้นๆ
    - **Province** จังหวัดของ Program นั้นๆ
    - **Language** ภาษาที่ใช้ใน Program นั้นๆ
    - **Package Availability** วันเปิดให้บริการของ Program
    - **Option** หมายถึง ช่องทางเสริมหรือโปรแกรมเสริมจากโปรแกรมหลัก เป็นส่วนที่กรอกหรือไม่กรอกก็ได้ เช่น โปรแกรม 1 มี Option เป็น บวกอาหารกลางวัน เป็นต้น

## **Itinerary** 
    หน้าต่างนี้กรอกเกี่ยวกับข้อมูลรายละเอียด Trip ของ Program
    - **Trip Round** กำหนดช่วงเวลารอบ Trip นั้นๆ
    - **Trip Type** แบ่งออกเป็น 2 ประเภท
        - **Join In** คือ ประเภทการเข้าร่วมกับกลุ่มอื่นๆ โดยเมื่อกดเลือก _Join In_ แล้ว จะแสดง _Minimum to run_ หากต้องการกำหนดจำนวนคนที่ต่ำที่สุดที่จะออกเดินทาง
        และมีรายละเอียดที่ต้องกรอกข้อมูลเพิ่มเติม ดังนี้
            - **Meet up point / Pick up point** โดยมีให้เลือกอย่างใดอย่างหนึ่ง และกรอกเพื่อกำหนดสถานที่นัดพบ
                - _Meet up point_ หมายถึงจุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร
                - _Pick up point_ หมายถึงจุดนัดรับที่จะเข้ารับผู้โดยสาร
            - **Start Trip** หมายถึง เวลาในการออกเดินทาง
                - _Detail_ รายละเอียดใน Start Trip
            - **Time** หมายถึง ช่วงเวลาที่เดินทางถึงสถานที่ท่องเที่ยว/ช่วงเวลาท่องเที่ยว
                - _Detail_ รายละเอียดในการท่องเที่ยวของช่วงเวลาที่กำหนด
                - _Add Time_ เพื่อใช้ในการเพิ่มช่วงเวลาท่องเที่ยว
            - **Drop off point** กำหนดช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
                - _End Trip_ ช่วงเวลาที่ถึงสถานที่จัดส่งผู้โดยสาร
                - _Drop off point_ สถานที่ที่จัดส่งผู้โดยสาร
                - _Add Drop off_ เพื่อใช้ในการเพิ่มช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
            
        - **Private** คือ ประเภทการ Trip แบบกลุ่มเดียวหรือ ส่วนตัว มีรายละเอียดที่ต้องกรอกข้อมูลเพิ่มเติม ดังนี้
            - **Start Trip** หมายถึง เวลาในการออกเดินทาง
                - _Detail_ รายละเอียดใน Start Trip
            - **Time** หมายถึง ช่วงเวลาที่เดินทางถึงสถานที่ท่องเที่ยว/ช่วงเวลาท่องเที่ยว
                - _Detail_ รายละเอียดในการท่องเที่ยวของช่วงเวลาที่กำหนด
                - _Add Time_ เพื่อใช้ในการเพิ่มช่วงเวลาท่องเที่ยว
            - **Drop off point** กำหนดช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
                - _End Trip_ ช่วงเวลาที่ถึงสถานที่จัดส่งผู้โดยสาร
                - _Drop off point_ สถานที่ที่จัดส่งผู้โดยสาร
                - _Add Drop off_ เพื่อใช้ในการเพิ่มช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสา

## **Unit**
    _Unit_ เป็นหน่วยการจอง โดยค่าตั้งต้นที่มีมาให้คือ ชื่อ Infant ประเภท Infant (ประเภทนี้เป็นการเข้าหน้างานไม่ได้มีการจอง เช่น เด็กเล็ก เป็นต้น)
    โดยจะแบ่งเป็น Trip Type หากเลือกเข้ามาเป็น Join In และ Private ระบบจะขึ้นมาให้กรอกสองประเภท 

    โดยสามารถเลือก Create Unit ในหน้านี้ได้เลย
    
    สามารถ _Add Unit_ เพิ่มได้ เพื่อระบุ เช่น ขายเป็นกลุ่ม, ขายเดี่ยว หรือขายเด็ก 
  </TabItem>

## **Agent**
    _Agent_ เป็นการระบุว่า Agent ใดใช้โปรแกรมนี้บ้าง (ยังไม่ต้องใส่ได้เช่นกัน) หากไม่มีการสร้าง Agent ก่อนจะไม่มีข้อมูลปรากฏขึ้น
  </TabItem>

## **Agent Quantity**
แสดงจำนวนของ Agent ที่มีการใช้งาน เมื่อนำเม้าส์ชี้ จะแสดงรายชื่อ Agent ที่มีการผูกกับ Program นั้นๆ

---

## **Status**
แสดงบอกสถานะ เปิด-ปิด การใช้งานสิทธิการเข้าถึงนั้นๆ โดยแบ่งออกเป็น 2 สถานะดังนี้
- **Active** บ่งบอกถึงสถานะนี้ _เปิดการใช้งาน_ สิทธิการเข้าถึง แสดงสีเป็นสีเขียว
- **Inactivate** บ่งบอกถึงสถานะนี้ _ปิดการใช้งาน_ สิทธิการเข้าถึง แสดงสีเป็นสีแดง

---

## **Last Modify Date**
แสดงวันที่ที่ทำการสร้างหรือแก้ไขข้อมูลล่าสุด

---

## **Action**
สามารถจัดการข้อมูลด้วยคำสั่งต่างๆ ได้ดังรูป
    - **View**
เพื่อเข้าตรวจสอบข้อมูลที่มีการสร้างขึ้นหรือแก้ไขล่าสุดแต่จะ _ไม่สามารถแก้ไขข้อมูล_  หากต้องการแก้ไขข้อมูลสามารถกด **Edit** ได้ 

    - **Delete**
เพื่อใช้ในการลบข้อมูล Program ที่ไม่ได้ใช้งานแล้วหรือต้องการลบออกจากระบบ
