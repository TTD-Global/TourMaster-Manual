# Program List

**Program List** คือรายละเอียด Product หรือรายละเอียด Program ของ Tour นั้นๆ

---

## ตารางรายการ Program

![Program List](/img/program/program-list.png)

</div>

ตารางแสดงรายการ Program ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อ Program |
| Seller Quantity | จำนวน Seller ที่ผูกกับ Program นั้นๆ |
| Status | สถานะการใช้งาน  |
| | **Draft**: ร่างข้อมูล ยังไม่สามารถใช้งานได้ |
| | <span class="badge badge-active">Active</span> เปิดใช้งาน |
| | <span class="badge badge-inactive">Inactive</span> ปิดใช้งาน |
| Last Modify Date | วันที่สร้างหรือแก้ไขล่าสุด |
| Action | ปุ่มจัดการสำหรับการ View / Edit |

:::caution
ในโหมด View จะ**ไม่สามารถแก้ไขข้อมูล**ได้ — หากต้องการแก้ไขให้กดปุ่ม **Edit** ภายในหน้า View
:::

## Program Detail

กรอกข้อมูลเบื้องต้นของ Program

![Program Detail](/img/program/program-detail.png)

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Program Name** | ชื่อของ Program tour นั้นๆ |
| **Destination** | จังหวัดของ Program นั้นๆ |
| **Service Language** | ภาษาที่ให้บริการใน Program นั้นๆ |
| **Package Availability** | วันที่เปิดขาย / วันเปิดให้บริการของ Program |
| **Arrangement Type** | ประเภทการจัดโปรแกรม (ดูรายละเอียดด้านล่าง) |
| **Booking Management** | รูปแบบการยืนยัน Booking |
| **Option** | แพ็กเกจเสริมหรือโปรแกรมเสริมจากโปรแกรมหลัก (กรอกหรือไม่กรอกก็ได้) เช่น เก็บค่าธรรมเนียมอุทยาน หรือ รวมค่าธรรมเนียมอุทยาน |

### Arrangement Type

เลือกประเภทการจัดโปรแกรม มี 3 ตัวเลือก

| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|----------|---------|
| **Excursion** | กิจกรรมหรือโปรแกรมท่องเที่ยว (เป็นได้ทั้งเรือและรถ) | Safari เป็นต้น |
| **Transport** | จัดรถรับ-ส่ง | ไปรับส่งสนามบิน-โรงแรม เป็นต้น |
| **Excursion + Transport** | รถรับ-ส่งเพื่อไปทำกิจกรรม หลังจบกิจกรรมจะนำลูกค้ากลับจุดเดิมหรือจุดที่ลูกค้าแจ้ง | รับลูกค้าจากจุดหนึ่งไปอีกจุดเพื่อทำกิจกรรม เป็นต้น |

### Booking Management

| ตัวเลือก | คำอธิบาย |
|---------|----------|
| **Auto Confirm** | เมื่อสร้าง Booking จะ Confirm อัตโนมัติ |
| **Manual** | เมื่อสร้าง Booking ต้องการกด Confirm ด้วยตนเอง |

## Itinerary

กรอกรายละเอียด Trip ของ Program

![Itinerary](/img/program/program-itinerary.png)

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Trip Round** | รอบกิจกรรม / ช่วงเวลารอบ Trip นั้นๆ |
| **Trip Type** | ประเภทการเดินทาง แบ่งเป็น **Join In** และ **Private** |
| **Meeting Point** | จุดนัดพบ เช่น ท่าเรือ หรือสถานที่ต่างๆ |
| **Itinerary** | รายละเอียดโปรแกรมทัวร์ |
| **Preview** | ตรวจสอบข้อมูลหลังจากกรอกเรียบร้อยแล้ว |

### Trip Type: Join In

Join In คือประเภทการเข้าร่วมกับกลุ่มอื่นๆ (รวมกลุ่ม)

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Minimum to run** | จำนวนลูกค้าขั้นต่ำสำหรับเริ่มต้นกิจกรรมนั้นๆ (กรอกหรือไม่กรอกก็ได้) |
| **Meeting point** | จุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร |
| **Start Trip** | เวลาออกเดินทาง พร้อม _Detail_ รายละเอียดเพิ่มเติม |
| **Time** | ช่วงเวลาท่องเที่ยว พร้อม _Detail_ และปุ่ม **Add Time** เพื่อเพิ่มช่วงเวลา |
| **Drop off point** | จุดส่งผู้โดยสาร ประกอบด้วย _End Trip_, _Drop off point_ และปุ่ม **Add Drop off** |



### Trip Type: Private

Private คือประเภทการ Trip แบบกลุ่มเดียวหรือส่วนตัว

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Meeting point** | จุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร |
| **Start Trip** | เวลาออกเดินทาง พร้อม _Detail_ รายละเอียดเพิ่มเติม |
| **Time** | ช่วงเวลาท่องเที่ยว พร้อม _Detail_ และปุ่ม **Add Time** เพื่อเพิ่มช่วงเวลา |
| **Drop off point** | จุดส่งผู้โดยสาร ประกอบด้วย _End Trip_, _Drop off point_ และปุ่ม **Add Drop off** |

## Cost Details

ค่าใช้จ่ายใน Program แบ่งออกเป็น 5 ส่วน ดังนี้

![Cost Details](/img/program/program-cost-header.png)

:::tip
หาก Itinerary เลือก Trip Type ทั้ง 2 ประเภท ส่วน Cost จะแสดงทั้ง 2 ประเภทด้วยเช่นกัน พร้อมปุ่ม **Copy** เพื่อคัดลอกข้อมูลจาก Trip Type หนึ่งไปยังอีกประเภท
:::

**Payment Option** มี 2 ตัวเลือก

| ตัวเลือก | คำอธิบาย |
|---------|----------|
| **Cash** | เงินสด |
| **Credit** | เครดิต |

**Cost Type** มี 3 ประเภท

| ประเภท | คำอธิบาย |
|--------|----------|
| **All** | รวมทั้งหมด (ราคาเหมา) |
| **Person** | ต่อคน |
| **Fix** | คิดตามจำนวนที่ระบุ โดยมี **Min (Pax)** และ **Max (Pax)** ช่วงจำนวนคน (ผู้เดินทาง) ต่ำสุด–สูงสุด สำหรับเรทราคานั้นๆ |

![Cost Type (Fix)](/img/program/program-cost-type.png)

:::info ตัวอย่าง Fix Cost
- Min Pax = 1, Max Pax = 10, Cost = 100 → กลุ่ม 1–10 คน คิดราคา 100 บาท
- กด **Add Fix Cost** เพื่อเพิ่มช่วงถัดไป → Min Pax = 11, Max Pax = 20, Cost = 200
:::

:::caution
- หากเลือก **Included** จะต้องกรอกข้อมูลให้ครบถ้วน หากไม่มีข้อมูลแนะนำให้เลือก **Not Included**
- หากทำการเลือกข้อมูลมาแล้วจะต้องกรอกข้อมูลให้ครบถ้วน ไม่เช่นนั้นจะบันทึกไม่ได้
:::

### Vehicle Cost

**Vehicle Cost** คือค่าใช้จ่ายยานพาหนะ

![Vehicle Cost](/img/program/program-vehicle-header.png)

กดปุ่ม **Add Vehicle** เพื่อเลือก Supplier หรือ Own Vehicle ที่มีการสร้างไว้

#### กรณีเลือก Supplier

![Supplier Selection](/img/program/program-vehicle-supplier-popup.png)

ระบบจะกรอง Supplier และ Group Type ตาม Arrangement Type ของ Program อัตโนมัติ

| Arrangement Type | Supplier / Group Type ที่แสดง |
|-----------------|------------------------------|
| **Excursion** | Excursion และ Excursion+Transport |
| **Transport** | Transport และ Excursion+Transport |
| **Excursion+Transport** | Excursion, Transport และ Excursion+Transport |

เมื่อเลือก Supplier และ Group Type แล้วจะแสดงตามตัวอย่าง

![Vehicle Supplier Result](/img/program/program-vehicle-supplier-result.png)

#### กรณีเลือก Own Vehicle

![Own Vehicle Selection](/img/program/program-vehicle-ownvehicle-popup.png)

ระบบจะกรอง Group Type ตาม Arrangement Type เช่นเดียวกับ Supplier

เมื่อเลือก Group Type ของ Own Vehicle แล้วจะต้องเลือก **Item Type** เพื่อระบุว่าเป็นค่าอะไร เช่น ค่าน้ำมัน 91, ค่าที่จอดรถ เป็นต้น

![Own Vehicle Result](/img/program/program-vehicle-ownvehicle-result.png)

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Payment Option** | _Cash_ — เงินสด / _Credit_ — เครดิต |
| **Supplier / Own Vehicle** | เลือก Supplier หรือ Own Vehicle |
| **Vehicle (Group Type)** | ข้อมูลจาก Group Type ของ Vehicle |
| **Option** | โปรแกรมเสริมที่ระบุใน Program Detail (หากไม่ระบุจะแสดง "–") |
| **Cost (THB)** | ระบุค่าใช้จ่ายเป็นตัวเลขทศนิยม |

</TabItem>

## Unit

**Unit** คือตัวเลือกการจอง โดยค่าตั้งต้นที่มีมาให้คือ _Infant (Default)_ ประเภท Infant (ประเภทนี้เป็นการเข้าหน้างานโดยไม่มีการจอง เช่น เด็กเล็ก)

![Unit](/img/program/program-unit-header.png)

กดปุ่ม **Add Unit** เพื่อเลือก Unit ตามที่มีการสร้างไว้ หากไม่มีสามารถกด **Create New Unit** ได้

![Add Unit](/img/program/program-unit-add-popup.png)

![Unit Form](/img/program/program-unit-form.png)

**Unit Name** คือชื่อที่ใช้เรียกสำหรับตัวเลือกการจองนั้นๆ

**Type** มี 2 ประเภท

| Type | คำอธิบาย |
|------|----------|
| **Person** | บุคคล — นับจำนวนคน |
| **Option** | ตัวเลือกเสริม — ไม่นับจำนวนคน เช่น ค่าธรรมเนียมอุทยานเป็นตัวเลือก |

![Unit Result](/img/program/program-unit-result.png)

:::info
- หากเลือก Trip Type ทั้ง Join In และ Private ระบบจะแสดงฟิลด์ Unit ทั้ง 2 ประเภท
:::

## Seller

**Seller** คือตัวแทนขาย

![Seller](/img/program/program-seller-header.png)

กดปุ่ม **Add Seller** เพื่อเลือก Seller ตามที่มีการสร้างไว้ หากไม่มีสามารถกด **Create Seller** ได้

![Add Seller](/img/program/program-seller-add-popup.png)

![Seller Form](/img/program/program-seller-form.png)

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Seller Name** | ชื่อตัวแทนขาย |
| **Booking No.** | หากเลือก **Auto Booking No.** ระบบจะสร้างหมายเลข Booking ให้อัตโนมัติ (สามารถแก้ไขได้) หากไม่เลือกต้องกรอกหมายเลขเอง |
| **Time Date Detail** | ช่วงวันที่ให้บริการ (From / End) |
| **Unit Detail & Selling Price** | กำหนดราคาขายแยกตาม Unit |

เมื่อเลือก Seller แล้วจะแสดงตามตัวอย่าง (หากยังไม่ต้องการกรอกข้อมูลสามารถกดย่อที่ลูกศรได้)

![Seller Result](/img/program/program-seller-result.png)

:::info
หากยังไม่มีการสร้าง Seller ไว้ในระบบ จะไม่มีข้อมูลให้เลือก ยังไม่ต้องใส่ในขั้นตอนนี้ก็ได้
:::
