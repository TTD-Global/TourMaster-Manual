# Program List

**Program List** คือรายการโปรแกรมทัวร์ทั้งหมดที่มีในระบบ ใช้สำหรับเลือกตอนสร้าง Booking และกำหนดราคา Unit สำหรับลูกค้าแต่ละประเภท

---

## ภาพรวม

ตาราง Program List แสดงรายการโปรแกรมทั้งหมด ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Program Name | ชื่อโปรแกรมทัวร์ |
| Program Code | รหัสโปรแกรม |
| Trip Type | ประเภทของ Trip |
| Duration | ระยะเวลา |
| Status | สถานะ (Active / Inactive) |
| Last Modify | วันที่แก้ไขล่าสุด |
| Action | ปุ่มจัดการ (View / Edit / Delete) |

---

## การสร้าง Program ใหม่

1. กดปุ่ม **Create**
2. กรอกข้อมูลโปรแกรม

### ข้อมูลที่ต้องกรอก

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|--------|
| **Program Name** | ชื่อโปรแกรมทัวร์ | ✅ |
| **Program Code** | รหัสโปรแกรม (ตัวอักษรหรือตัวเลข) | ✅ |
| **Trip Type** | ประเภท Trip (Half Day / Full Day / ฯลฯ) | ✅ |
| **Duration** | ระยะเวลาของโปรแกรม | — |
| **Description** | คำอธิบายโปรแกรม | — |

3. ตั้งค่า **Unit & Pricing** — กำหนดราคาสำหรับแต่ละ Unit Type และกลุ่มลูกค้า
4. กดปุ่ม **Save** เพื่อบันทึก

---

## การจัดการ Program

- **View** — ดูรายละเอียดโปรแกรมแบบอ่านอย่างเดียว
- **Edit** — แก้ไขข้อมูลโปรแกรม เช่น ชื่อ ราคา หรือ Unit
- **Delete** — ลบโปรแกรมที่ไม่ใช้แล้ว (ไม่สามารถลบได้หากมี Booking ที่ใช้งานอยู่)

---

## การเปิด/ปิด Status

- **Active** — โปรแกรมพร้อมใช้งาน สามารถเลือกได้ตอนสร้าง Booking
- **Inactive** — ปิดการใช้งานชั่วคราว ไม่แสดงให้เลือกในหน้า Create Booking

:::tip
ควรตั้งค่า Item Type และ Unit ก่อนสร้าง Program เพื่อให้สามารถกำหนดราคาได้อย่างถูกต้อง
:::
