# Advance

**Advance** คือหน้าจัดการเงินเบิกล่วงหน้าสำหรับ Guide และ Driver แต่ละ Trip ก่อนออกเดินทาง

เข้าถึงได้จาก **Payment → Advance**

---

## ตารางรายการ

![Advance](/img/Payment/payment-advance.png)

### Summary Card

ด้านบนแสดง Card สรุปภาพรวม

| Card | คำอธิบาย |
|------|----------|
| **Amount Trip** | จำนวน Trip ทั้งหมด |
| **Total Advance** | ยอดเงิน Advance รวม |
| **Total Extra Advance** | ยอดเงิน Extra Advance รวม |
| **Total** | ยอดรวมทั้งหมด (Advance + Extra Advance) |

### แท็บสถานะ

| แท็บ | คำอธิบาย |
|------|----------|
| **Pending** | Trip ที่ยังรอดำเนินการ |
| **Completed** | Trip ที่ดำเนินการเสร็จแล้ว |

### คอลัมน์ตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Trip Code** | รหัส Trip |
| **Travel Date** | วันที่เดินทาง |
| **Trip Type** | ประเภท Trip (เช่น Join In) |
| **Trip Round** | รอบเวลาเดินทาง |
| **Program** | ชื่อโปรแกรม |
| **Guide** | ชื่อ Guide ที่รับผิดชอบ |
| **Bank Detail** | ข้อมูลบัญชีธนาคารของ Guide (เลขบัญชี, ชื่อเจ้าของบัญชี) |
| **Pax** | จำนวนผู้เดินทาง |
| **Advance** | ยอดเงินเบิกล่วงหน้า |
| **Extra Advance** | ยอดเงินเบิกเพิ่มเติม |
| **Total** | ยอดรวม (Advance + Extra Advance) |
| **Action** | ปุ่มจัดการ |

แถวสุดท้ายแสดง **Total** รวม Advance, Extra Advance, Total ทั้งหมด

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Export** | ส่งออกรายการ Advance เป็นไฟล์ |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อจัดการรายการ

### View

แสดงรายละเอียดรายการ Advance ของ Trip นั้น รวมถึงรายการค่าใช้จ่ายแยกตามประเภท

:::info
Advance จะถูกนำไปเปรียบเทียบกับค่าใช้จ่ายจริงใน **Expenses** และสรุปยอดคงเหลือใน **Balance**
:::
