# Expenses

**Expenses** คือหน้าบันทึกค่าใช้จ่ายที่เกิดขึ้นจริงระหว่างการดำเนินทริป เปรียบเทียบกับเงิน Advance ที่เบิกไว้

เข้าถึงได้จาก **Payment → Expenses**

---

## ตารางรายการ

![Expenses](/img/Payment/payment-expenses.png)

### Summary Card

ด้านบนแสดง Card สรุปภาพรวม

| Card | คำอธิบาย |
|------|----------|
| **Amount Trip** | จำนวน Trip ทั้งหมด |
| **Total Advance** | ยอดเงิน Advance รวม |
| **Total Expense** | ยอดค่าใช้จ่ายจริงรวม |
| **Total** | ยอดรวมทั้งหมด |

### แท็บสถานะ

| แท็บ | คำอธิบาย |
|------|----------|
| **Pending** | Trip ที่ยังรอดำเนินการ |
| **Completed** | Trip ที่ดำเนินการเสร็จแล้ว |

### คอลัมน์ตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Trip Code** | รหัส Trip |
| **Trip Type** | ประเภท Trip (เช่น Join In) |
| **Trip Round** | รอบเวลาเดินทาง |
| **Program** | ชื่อโปรแกรม |
| **Guide** | ชื่อ Guide ที่รับผิดชอบ |
| **Bank Detail** | ข้อมูลบัญชีธนาคารของ Guide |
| **Pax** | จำนวนผู้เดินทาง |
| **Advance** | ยอดเงินเบิกล่วงหน้า |
| **Expense** | ยอดค่าใช้จ่ายจริง |
| **Action** | ปุ่มจัดการ |

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Date Filter** | กรองรายการตามวันที่ |
| **Export** | ส่งออกรายการ Expenses เป็นไฟล์ |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อจัดการรายการ

### View

แสดงรายละเอียดค่าใช้จ่ายจริงของ Trip นั้น แยกตามรายการ

:::tip เคล็ดลับ
ระบบจะนำข้อมูล Expenses ไปคำนวณยอดคงเหลือให้อัตโนมัติใน **Balance**
:::
