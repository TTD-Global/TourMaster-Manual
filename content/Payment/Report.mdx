# Report

**Report** คือรายงานสรุปค่าใช้จ่ายทั้งหมดของ Payment แยกตามประเภท เพื่อใช้ติดตามและวิเคราะห์ต้นทุนของแต่ละ Trip ในช่วงเวลาที่เลือก

เข้าถึงได้จาก **Payment → Report**

---

## ภาพรวม

![Payment Report](/img/Payment/payment-report.png)

### Summary Card

ด้านบนแสดง Card สรุปยอดรวมของเดือนที่เลือก

| Card | คำอธิบาย |
|------|----------|
| **Total Expense** | ยอดค่าใช้จ่ายรวมทั้งหมด |
| **Supplier Cost** | ค่าใช้จ่าย Supplier รวม |
| **Own Vehicle Cost** | ค่าใช้จ่ายรถบริษัทรวม |
| **Guide Cost** | ค่าใช้จ่าย Guide รวม |

---

## ประเภทรายงาน

รายงานแบ่งออกเป็น 4 ประเภท เลือกได้จากแท็บด้านบน

| แท็บ | คำอธิบาย |
|------|----------|
| **Expense** | รายงานค่าใช้จ่ายรวมทุกประเภท แยกตาม: All, Guide fee, Vehicle Cost, Other Expense, Allowance, Extra Cost, Expense Cost |
| **Supplier** | รายงานค่าใช้จ่ายของ Supplier |
| **Own Vehicle** | รายงานค่าใช้จ่ายรถบริษัทเอง |
| **Guide** | รายงานค่าใช้จ่าย Guide |

### คอลัมน์ตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Type** | ประเภทค่าใช้จ่าย |
| **List** | รายการย่อย |
| **Total Trips** | จำนวน Trip ที่มีรายการนี้ |
| **Total (Baht)** | ยอดรวมเป็นบาท |

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Date Range Filter** | เลือกช่วงวันที่ (ค่าเริ่มต้น: ต้นเดือน–สิ้นเดือนปัจจุบัน) |
| **Export** | ดาวน์โหลดรายงานเป็นไฟล์ |

---

## วิธีดูรายงาน

1. เลือกช่วงวันที่จาก **Date Range Filter**
2. เลือกประเภทรายงานจากแท็บ (**Expense / Supplier / Own Vehicle / Guide**)
3. เลือก Sub-tab เพื่อดูรายละเอียดเพิ่มเติม (กรณีแท็บ Expense)
4. ดูสรุปจาก Summary Card และตาราง
5. กดปุ่ม **Export** เพื่อดาวน์โหลดรายงาน

:::tip เคล็ดลับ
ใช้ Report ร่วมกับ **Balance** เพื่อตรวจสอบและปิดบัญชีค่าใช้จ่ายประจำเดือน
:::
