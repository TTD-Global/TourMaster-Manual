# Booking List

**Booking List** คือหน้าสำหรับดูและจัดการรายการ Booking ทั้งหมดในระบบ

---

## ตารางรายการ

![Booking List](/img/Bookings/booking-list.png)

### Status Card

ด้านบนตารางแสดงจำนวน Booking แยกตามกลุ่มสถานะ สามารถกดที่ Card เพื่อกรองตารางตามสถานะนั้นได้ทันที

**In Progress** — Booking ที่อยู่ระหว่างดำเนินการ

| สถานะ | คำอธิบาย |
|-------|----------|
| **Reserved** | Booking ที่สร้างแล้ว รอการยืนยัน |
| **Confirmed** | Booking ที่ได้รับการยืนยันแล้ว |

**Processed** — Booking ที่ดำเนินการเสร็จสิ้น (กรองตามเดือนปัจจุบัน)

| สถานะ | คำอธิบาย |
|-------|----------|
| **Completed** | Booking ที่เสร็จสมบูรณ์ |
| **Canceled** | Booking ที่ถูกยกเลิก |

### คอลัมน์ตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Seller** | ชื่อ Seller หรือตัวแทนที่รับจอง |
| **Travel Date** | วันที่เดินทาง |
| **Booking Date** | วันที่สร้าง Booking |
| **Booking No.** | เลขที่ Booking (คลิกได้เพื่อดูรายละเอียด) |
| **Trip Type** | ประเภทของ Trip |
| **Travel Time** | เวลาเดินทาง |
| **Program** | ชื่อโปรแกรมทัวร์ |
| **Pax** | จำนวนผู้เดินทาง |
| **Trip Code** | รหัส Trip |
| **Status** | สถานะปัจจุบัน |
| **Action** | ปุ่มจัดการ |

## Action

คลิกที่ **Booking No.** หรือกดปุ่มในคอลัมน์ **Action** เพื่อจัดการ Booking

### View

แสดงรายละเอียด Booking แบบ read-only

- กดปุ่ม **Edit** เพื่อเข้าสู่โหมดแก้ไข

### Edit

แก้ไขข้อมูล Booking ที่มีสถานะ **Reserved** หรือ **Confirmed**

1. แก้ไขข้อมูลในส่วนที่ต้องการ
2. กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

:::caution ข้อควรระวัง
Booking ที่มีสถานะ **Completed** จะไม่สามารถแก้ไขได้
:::

### Cancel

ยกเลิก Booking — สถานะจะเปลี่ยนเป็น **Canceled**

:::caution ข้อควรระวัง
การยกเลิก Booking ไม่สามารถกู้คืนได้ ควรตรวจสอบให้แน่ใจก่อนดำเนินการ
:::
