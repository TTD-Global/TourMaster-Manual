# Company Overview

**Company Overview** คือ Dashboard ภาพรวมของบริษัท แสดงสรุปรายได้ ค่าใช้จ่าย กำไร/ขาดทุน และสถิติการดำเนินงานทั้งหมดในช่วงเวลาที่เลือก

เข้าถึงได้จาก **Dashboard → Company Overview**

---

## ภาพรวม

<div className="manual-img-500">

![Company Overview](/img/Dashboard/dashboard-company-overview.png)

</div>

### ตัวกรอง

| ตัวกรอง | คำอธิบาย |
|---------|----------|
| **All Destinations** | กรองตามจุดหมายปลายทาง |
| **All Programs** | กรองตามโปรแกรมทัวร์ |
| **แท็บช่วงเวลา** | Today / Weekly / Monthly / Yearly |

### แท็บมุมมอง

| แท็บ | คำอธิบาย |
|------|----------|
| **Overview** | ภาพรวมทั้งหมด — Summary Cards, กราฟ, Program Performance |
| **Seller** | สรุปยอดขายและรายได้แยกตาม Seller |
| **Expense** | สรุปค่าใช้จ่ายแยกตามประเภท |

---

## Summary Cards

ด้านบนแสดง Card สรุปตัวเลขสำคัญของวันที่เลือก

| Card | คำอธิบาย |
|------|----------|
| **Total Revenue** | รายได้รวมทั้งหมด |
| **Total Expenses** | ค่าใช้จ่ายรวมทั้งหมด |
| **Total Profit/Loss** | กำไร/ขาดทุนสุทธิ (Revenue − Expenses) พร้อม % |
| **Total Advances** | ยอดเงิน Advance รวม |
| **Total Participants** | จำนวนผู้เดินทางรวม |
| **Check In** | จำนวนผู้ที่ Check In แล้ว |
| **No Show** | จำนวนผู้ที่ไม่มาตามนัด |
| **Total Trip** | จำนวน Trip ทั้งหมด (แยก Join In / Private) |
| **Total Guide** | จำนวน Guide ทั้งหมด (แยก Guide / Staff) |
| **Total Vehicle** | จำนวนยานพาหนะ (แยก Supplier / Own Vehicle) |
| **Total Guide Expenses** | ค่าใช้จ่าย Guide รวม |
| **Total Vehicle Expenses** | ค่าใช้จ่ายยานพาหนะรวม |

## Program Performance

<div className="manual-img-500">

![Program Performance](/img/Dashboard/dashboard-company-program.png)

</div>

ตารางสรุปผลการดำเนินงานแยกตามโปรแกรม

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **Program List** | ชื่อโปรแกรมทัวร์ |
| **Trips** | จำนวน Trip ที่วิ่งโปรแกรมนี้ |
| **Total Pax** | จำนวนผู้เดินทางรวม |
| **Revenue (฿)** | รายได้รวม |
| **Expenses (฿)** | ค่าใช้จ่ายรวม |
| **Profit/Loss (฿)** | กำไร/ขาดทุน |

กดปุ่ม **View All** เพื่อดูรายการโปรแกรมทั้งหมด
