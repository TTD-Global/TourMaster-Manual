# Guide

**Guide Dashboard** คือ Dashboard ติดตามสถิติการทำงานของ Guide และ Staff แสดงจำนวน Guide ที่ใช้งาน, Trip ที่รับผิดชอบ และ Ranking การมอบหมายงาน

เข้าถึงได้จาก **Dashboard → Guide**

### ตัวกรองช่วงเวลา

| แท็บ | คำอธิบาย |
|------|----------|
| **Today** | ข้อมูลวันนี้ |
| **7 Days** | ข้อมูล 7 วันย้อนหลัง |
| **30 Days** | ข้อมูล 30 วันย้อนหลัง |
| **12 Months** | ข้อมูล 12 เดือนย้อนหลัง |

---

## Summary Cards

| Card | คำอธิบาย |
|------|----------|
| **Total Guide** | จำนวน Guide ทั้งหมดในระบบ (แยก Guide / Staff) |
| **Guide & Staff used** | จำนวน Guide และ Staff ที่ถูกมอบหมายงานในช่วงนั้น (แยก Guide / Staff) พร้อม % |
| **Total Trip** | จำนวน Trip ทั้งหมด (แยก Private Trip / Join In Trip) พร้อม % |
| **Total Participants** | จำนวนผู้เดินทางรวม พร้อม % |

| ส่วน | คำอธิบาย |
|------|----------|
| **Guide usage by program** | กราฟแสดงการใช้งาน Guide แยกตามโปรแกรม กดปุ่ม **View all programs** เพื่อดูทั้งหมด |
| **Top guides assigned most tasks** | อันดับ Guide ที่ได้รับมอบหมายงานมากที่สุด กดปุ่ม **View guide ranking** เพื่อดูรายละเอียด |
