# Supplier List

หน้า **Supplier List** ใช้จัดการรายชื่อผู้ให้บริการยานพาหนะภายนอก (Supplier) ที่บริษัทใช้งาน

เข้าถึงได้จาก **Setting → Vehicle → Supplier List**

---

## ตารางรายการ

![Supplier List](/img/Vehicle/supplier-list.png)

ตารางแสดงรายการ Supplier ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Image** | รูปโปรไฟล์ |
| **Supplier Name** | ชื่อ Supplier (คลิกได้เพื่อดูรายละเอียด) |
| **Group Type quantity** | จำนวนประเภทรถที่ผูกไว้ (คลิกได้) |
| **Status** | สถานะการใช้งาน |
| | **Active**: เปิดใช้งาน (Badge สีเขียว) |
| | **Inactive**: ปิดใช้งาน (Badge สีแดง) |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อเลือกคำสั่ง หรือคลิกที่ชื่อ Supplier เพื่อดูรายละเอียด

:::caution ข้อควรระวัง
ในโหมด View จะ**ไม่สามารถแก้ไขข้อมูล**ได้ — หากต้องการแก้ไขให้กดปุ่ม **Edit** ภายในหน้า View
:::

### View

แสดงข้อมูล Supplier แบบ read-only ประกอบด้วย 4 แท็บ ได้แก่ **General Detail**, **Supplier Login**, **Registration** และ **Personnel list**

- กดปุ่ม **Edit** เพื่อเข้าสู่โหมดแก้ไข
- **Status** — เปลี่ยนสถานะ Active / Inactive ได้จากหน้า View โดยตรง

### Edit

แก้ไขข้อมูล Supplier ได้ทุกส่วน รวมถึง Reset Password ของ Supplier Login

1. แก้ไขข้อมูลในฟิลด์ที่ต้องการ
2. กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

### Delete

กดเพื่อลบ Supplier ออกจากระบบ

:::caution ข้อควรระวัง
การลบ Supplier จะไม่สามารถกู้คืนได้
:::

### Supplier Login

สร้างบัญชีสำหรับ Supplier Login เพื่อให้ Driver ของ Supplier เข้าสู่ระบบตรวจสอบงานได้

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|:------:|
| **Username** | ชื่อบัญชีเข้าสู่ระบบ | ✅ |
| **Password** | รหัสผ่าน | ✅ |
| **Confirm Password** | ยืนยันรหัสผ่าน | ✅ |

กด **Save** เพื่อบันทึกข้อมูลทั้งหมดและสร้าง Supplier

:::tip เคล็ดลับ
สามารถกด **Save and exit** ในขั้นตอนที่ 1 เพื่อบันทึกข้อมูลทั่วไปก่อน แล้วกลับมาสร้าง Supplier Login ภายหลังได้
:::

</TabItem>
