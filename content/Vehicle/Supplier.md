# Supplier List

**Supplier List** คือรายการบริษัทหรือผู้ให้บริการยานพาหนะภายนอก (Supplier) ที่บริษัทใช้บริการ ข้อมูลนี้จะนำไปใช้ในการจัดสรรรถให้กับ Excursion / Transport ต่างๆ

---

## ภาพรวม

ตาราง Supplier List แสดงรายการ Supplier ทั้งหมด

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Supplier Name | ชื่อบริษัท/ผู้ให้บริการ |
| Contact | ข้อมูลการติดต่อ |
| Phone | เบอร์โทรศัพท์ |
| Status | สถานะ (Active / Inactive) |
| Action | ปุ่มจัดการ (View / Edit / Delete) |

---

## การสร้าง Supplier ใหม่

1. กดปุ่ม **Create**
2. กรอกข้อมูล Supplier

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|--------|
| **Supplier Name** | ชื่อบริษัทหรือผู้ให้บริการ | ✅ |
| **Contact Person** | ชื่อผู้ติดต่อ | — |
| **Phone** | เบอร์โทรศัพท์ | — |
| **Email** | อีเมล | — |
| **Address** | ที่อยู่ | — |
| **Note** | หมายเหตุ | — |

3. กดปุ่ม **Save** เพื่อบันทึก

---

## การเพิ่มรถให้ Supplier

หลังจากสร้าง Supplier แล้ว สามารถเพิ่มรายการรถ (Vehicle) ที่ Supplier มีให้บริการได้

1. คลิก **View** หรือ **Edit** ที่ Supplier
2. ในส่วน **Vehicle List** กดปุ่ม **Add Vehicle**
3. กรอกข้อมูลรถ: ประเภทรถ, ทะเบียน, จำนวนที่นั่ง
4. กดบันทึก

:::tip
ข้อมูล Supplier ที่ครบถ้วนจะช่วยให้การจัดสรรรถใน Excursion / Transport ทำได้รวดเร็วและถูกต้องมากขึ้น
:::
