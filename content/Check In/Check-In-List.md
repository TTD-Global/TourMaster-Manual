# Check In List

**Check In List** คือรายงานสรุปรายชื่อลูกค้าที่ได้ทำการ Check In ในวันที่เลือก พร้อมข้อมูลสถานะและรายละเอียด Booking

---

## ภาพรวม

ตารางแสดงรายชื่อลูกค้าทั้งหมดที่ Check In แล้ว ประกอบด้วย

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Customer Name | ชื่อลูกค้า |
| Booking No. | เลขที่ Booking |
| Program | ชื่อโปรแกรม |
| Trip Type | ประเภท Trip |
| Pax | จำนวนผู้เดินทาง |
| Check In Time | เวลาที่ Check In |
| Status | สถานะ (Checked In / No Show) |
| Action | ปุ่มดูรายละเอียด |

---

## การกรองข้อมูล

- **Date Filter** — เลือกวันที่ที่ต้องการดูรายการ
- **Status Filter** — กรองตามสถานะ (Checked In, No Show)

---

## การ Export

กดปุ่ม **Export** เพื่อส่งออกรายชื่อลูกค้า Check In เป็นไฟล์ Excel สำหรับใช้อ้างอิง
