# Guide / Staff

**Guide / Staff** คือรายการพนักงานที่ทำหน้าที่เป็น Tour Guide หรือ Staff ในการดูแลลูกค้าระหว่างทริป

---

## ภาพรวม

ตาราง Guide/Staff แสดงรายชื่อพนักงานทั้งหมด

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| # | ลำดับที่ |
| Employee Code | รหัสพนักงาน |
| Name | ชื่อ-นามสกุล |
| Position | ตำแหน่งงาน |
| Phone | เบอร์โทรศัพท์ |
| Status | สถานะ (Active / Inactive) |
| Action | ปุ่มจัดการ |

---

## การเพิ่ม Guide / Staff

1. กดปุ่ม **Create**
2. กรอกข้อมูลพนักงาน

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|--------|
| **Employee Code** | รหัสพนักงาน | ✅ |
| **First Name** | ชื่อ | ✅ |
| **Last Name** | นามสกุล | ✅ |
| **Position** | ตำแหน่งงาน | ✅ |
| **Phone** | เบอร์โทรศัพท์ | — |
| **Email** | อีเมล | — |
| **Language** | ภาษาที่ใช้ได้ | — |

3. กดปุ่ม **Save** เพื่อบันทึก

:::tip
ควรสร้าง Position ไว้ในเมนู Employee > Positions ก่อน เพื่อให้มีข้อมูลให้เลือก
:::
