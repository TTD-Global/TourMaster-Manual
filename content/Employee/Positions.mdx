# Positions

หน้า **Positions** ใช้จัดการตำแหน่งงานของพนักงานในระบบ ข้อมูลที่สร้างที่นี่จะถูกนำไปใช้ในหน้า Guide / Staff และ Other

เข้าถึงได้จาก **Setting → Employee → Positions**

---

## ตารางรายการ

<div className="manual-img-500">

![Positions List](/img/Employee/positions-list.png)

</div>

ตารางแสดงรายการตำแหน่งงานทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Position List** | ชื่อตำแหน่งงาน |
| **Status** | สถานะการใช้งาน |
| | **Active**: เปิดใช้งาน (Badge สีเขียว) |
| | **Inactive**: ปิดใช้งาน (Badge สีแดง) |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

ตำแหน่งที่มีป้ายกำกับ **(Default)** คือตำแหน่งที่ระบบสร้างให้ตั้งแต่แรก ได้แก่ **Guide**, **Staff** และ **Driver** — ไม่สามารถลบได้

---

## ฟังก์ชันบนหน้า

| ปุ่ม | คำอธิบาย |
|------|----------|
| **Create Position** | สร้างตำแหน่งงานใหม่ |
| **Search** | ค้นหาตำแหน่ง |
| **Filter** | กรองข้อมูล |
| **Export** | ดาวน์โหลดรายการตำแหน่ง |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อจัดการข้อมูลตำแหน่ง

:::caution ข้อควรระวัง
ไม่สามารถลบตำแหน่ง Default (Guide, Staff, Driver) ได้  
และไม่ควรลบตำแหน่งที่มีพนักงานใช้งานอยู่
:::

### View

- แสดงชื่อตำแหน่งและสถานะแบบ read-only
- กดปุ่ม **Edit** เพื่อเข้าสู่โหมดแก้ไข

### Edit

- แก้ไขชื่อตำแหน่งงาน
- **Status** — เปลี่ยนสถานะระหว่าง **Active** และ **Inactive**
- กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

### Delete

กดเพื่อลบตำแหน่งงานที่ไม่ต้องการออกจากระบบ

---

## Create

กดปุ่ม **Create Position** แล้วกรอกข้อมูลในฟอร์ม

<div className="manual-img-500">

![Create Position](/img/Employee/positions-create.png)

</div>

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|:------:|
| **Position Name** | ชื่อตำแหน่งงาน | ✅ |

1. กรอกชื่อตำแหน่งงานในฟิลด์ **Position Name**
2. กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

:::tip เคล็ดลับ
ควรสร้าง Position ก่อนเพิ่มข้อมูลพนักงานในหน้า Guide / Staff หรือ Other เนื่องจากฟิลด์ Position เป็นข้อมูลจำเป็น
:::
