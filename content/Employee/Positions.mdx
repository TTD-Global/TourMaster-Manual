# Positions

หน้า **Positions** ใช้จัดการตำแหน่งงานของพนักงานในระบบ ข้อมูลที่สร้างที่นี่จะถูกนำไปใช้ในหน้า Guide/Staff และ Other

เข้าถึงได้จาก **Setting → Employee → Positions**

---

## ภาพรวม

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Position List** | ชื่อตำแหน่งงาน (คลิกได้เพื่อดูรายละเอียด) |
| **Status** | สถานะ: **Active** / **Inactive** |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

ตำแหน่งที่มี **( Default )** กำกับ คือตำแหน่งที่ระบบสร้างไว้ให้ตั้งแต่แรก ได้แก่ **Guide**, **Staff** และ **Driver**

---

## ฟังก์ชันบนหน้า

| ปุ่ม | คำอธิบาย |
|------|----------|
| **Create Position** | สร้างตำแหน่งงานใหม่ |
| **Search** | ค้นหาตำแหน่ง |
| **Filter** | กรองข้อมูล |
| **Export** | ดาวน์โหลดรายการตำแหน่ง |

---

## การสร้าง Position ใหม่

1. กดปุ่ม **Create Position**
2. กรอกชื่อตำแหน่งงาน
3. กดบันทึก

---

## การจัดการ Position

กดปุ่ม `...` ในคอลัมน์ Action เพื่อแก้ไขหรือเปลี่ยนสถานะตำแหน่งงาน

:::caution
ควรสร้าง Position ก่อนเพิ่มข้อมูลพนักงาน เนื่องจากฟิลด์ Position เป็นข้อมูลที่จำเป็น
:::
