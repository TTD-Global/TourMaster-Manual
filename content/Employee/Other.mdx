# Other

หน้า **Other** ใช้จัดการรายชื่อพนักงานประเภทอื่น ๆ ที่ไม่ใช่ Guide หรือ Staff เช่น Driver, IT Support หรือตำแหน่งที่สร้างเพิ่มเติมเอง

เข้าถึงได้จาก **Setting → Employee → Other**

---

## ภาพรวม

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Image** | รูปโปรไฟล์พนักงาน |
| **Position** | ตำแหน่งงาน |
| **Name** | ชื่อ-นามสกุลพนักงาน |
| **Phone Number** | เบอร์โทรศัพท์ |
| **Registration** | หมายเลขทะเบียน (เช่น ทะเบียนรถ) |
| **Status** | สถานะ: **Active** (เขียว) / **Inactive** |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

---

## ฟังก์ชันบนหน้า

| ปุ่ม | คำอธิบาย |
|------|----------|
| **Create Other** | สร้างพนักงานประเภท Other ใหม่ |
| **Search** | ค้นหาพนักงาน |
| **Filter** | กรองข้อมูลตามเงื่อนไข |
| **Export** | ดาวน์โหลดรายชื่อพนักงาน |

---

## การสร้างพนักงาน Other ใหม่

1. กดปุ่ม **Create Other**
2. กรอกข้อมูลพนักงาน เช่น ชื่อ, ตำแหน่ง, เบอร์โทร, หมายเลขทะเบียน
3. กดบันทึก

:::info
ตำแหน่ง (Position) ที่ใช้ในหน้า Other จะดึงมาจากข้อมูลที่สร้างไว้ใน **Employee → Positions** ควรสร้าง Position ก่อน
:::
