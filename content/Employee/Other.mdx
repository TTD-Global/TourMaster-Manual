# Other

หน้า **Other** ใช้จัดการรายชื่อพนักงานประเภทอื่นๆ ที่ไม่ใช่ Guide หรือ Staff เช่น Driver, IT Support หรือตำแหน่งที่สร้างขึ้นเพิ่มเติมเอง

เข้าถึงได้จาก **Setting → Employee → Other**

---

## ตารางรายการ

<div className="manual-img-500">

![Other List](/img/Employee/other-list.png)

</div>

ตารางแสดงรายการพนักงาน Other ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Image** | รูปโปรไฟล์พนักงาน |
| **Position** | ตำแหน่งงาน |
| **Name** | ชื่อ-นามสกุลพนักงาน |
| **Phone Number** | เบอร์โทรศัพท์ |
| **Registration** | ยานพาหนะที่มอบหมาย |
| **Status** | สถานะการใช้งาน |
| | <span class="badge badge-active">Active</span> เปิดใช้งาน |
| | <span class="badge badge-inactive">Inactive</span> ปิดใช้งาน |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อเลือกคำสั่ง

### View

แสดงข้อมูลพนักงานทั้งหมดแบบ read-only

- กดปุ่ม **Edit** ภายในหน้า View เพื่อเข้าสู่โหมดแก้ไข

#### Edit

- แก้ไขข้อมูลได้ทุกฟิลด์
- **Status** — เปลี่ยนสถานะระหว่าง **Active** และ **Inactive**
- กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก


:::caution ข้อควรระวัง
ในโหมด View จะ**ไม่สามารถแก้ไขข้อมูล**ได้ — หากต้องการแก้ไขให้กดปุ่ม **Edit** ภายในหน้า View
:::

## Create

กดปุ่ม **Create Other** แล้วกรอกข้อมูลในฟอร์ม

<div className="manual-img-500">

![Create Other](/img/Employee/other-create.png)

</div>

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|:------:|
| **Photo** | อัปโหลดรูปพนักงาน | — |
| **Position** | ตำแหน่งงาน (เลือกจาก Positions ที่สร้างไว้) | ✅ |
| **Name** | ชื่อ-นามสกุลพนักงาน | ✅ |
| **Phone Number** | เบอร์โทรศัพท์ | ✅ |
| **Email** | อีเมลพนักงาน | — |
| **Registration** | ยานพาหนะที่มอบหมาย | — |

1. กรอกข้อมูลในฟอร์ม
2. กด **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

:::info
ตำแหน่งงาน (Position) ที่ใช้ในหน้า Other จะดึงมาจากข้อมูลที่สร้างไว้ใน **Employee → Positions**  
ควรสร้าง Position ก่อนเพิ่มข้อมูลพนักงาน
:::
