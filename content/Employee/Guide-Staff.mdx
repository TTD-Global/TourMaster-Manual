# Guide / Staff

หน้า **Guide / Staff** ใช้จัดการรายชื่อพนักงานประเภท Guide และ Staff ภายในองค์กร

เข้าถึงได้จาก **Setting → Employee → Guide / Staff**

---

## ภาพรวม

หน้านี้แบ่งเป็น 2 แท็บ

| แท็บ | คำอธิบาย |
|------|----------|
| **Guide List** | รายการพนักงาน Guide ทั้งหมด |
| **Staff List** | รายการพนักงาน Staff ทั้งหมด |

---

## คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Image** | รูปโปรไฟล์พนักงาน |
| **Nickname** | ชื่อเล่น / รหัสย่อ |
| **Guide Name / Staff Name** | ชื่อ-นามสกุลพนักงาน (คลิกได้เพื่อดูรายละเอียด) |
| **Phone Number** | เบอร์โทรศัพท์ |
| **License No.** | หมายเลขใบอนุญาต |
| **Work Area** | จำนวนพื้นที่ทำงานที่มอบหมาย |
| **Language** | จำนวนภาษาที่ใช้งานได้ |
| **Status** | สถานะ: **Active** (เขียว) / **Draft** (เทา) |
| **Last Modify Date** | วันที่แก้ไขล่าสุด |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

---

## ฟังก์ชันบนหน้า

| ปุ่ม | คำอธิบาย |
|------|----------|
| **Create Guide / Create Staff** | สร้างพนักงานใหม่ (ปุ่มเปลี่ยนตามแท็บที่เลือก) |
| **Search** | ค้นหาพนักงาน |
| **Filter** | กรองข้อมูลตามเงื่อนไข |
| **Export** | ดาวน์โหลดรายชื่อพนักงาน |

---

## การสร้าง Guide / Staff ใหม่

กดปุ่ม **Create Guide** หรือ **Create Staff** แล้วทำตาม 3 ขั้นตอน

### ขั้นตอนที่ 1 — General Detail

กรอกข้อมูลทั่วไปของพนักงาน

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|----------|:------:|
| รูปโปรไฟล์ | อัปโหลดรูปพนักงาน | — |
| **Title** | คำนำหน้าชื่อ | ✅ |
| **First Name** | ชื่อ | ✅ |
| **Last Name** | นามสกุล | ✅ |
| **Nickname** | ชื่อเล่น / รหัสย่อ | ✅ |
| **ID Card Number** | หมายเลขบัตรประชาชน | ✅ |
| **Date of Birth** | วันเกิด | — |
| **Line ID** | ไลน์ไอดี | — |
| **Phone Number** | เบอร์โทรศัพท์ | ✅ |
| **Line User ID** | Line User ID (สำหรับ integration) | — |
| **Upload ID Card Number Photos** | รูปถ่ายบัตรประชาชน (JPG/PNG/HEIC/JPEG ไม่เกิน 5 MB) | ✅ |

### ขั้นตอนที่ 2 — Guide Detail

กรอกข้อมูลสำหรับการมอบหมายงาน Guide เช่น ภาษา, พื้นที่ทำงาน, ใบอนุญาต

### ขั้นตอนที่ 3 — Bank Detail

กรอกข้อมูลบัญชีธนาคารสำหรับการจ่ายค่าตอบแทน

:::tip
กดปุ่ม **Save and exit** เพื่อบันทึกข้อมูลในขั้นตอนปัจจุบันก่อน แล้วกลับมาแก้ไขต่อภายหลังได้
กดปุ่ม **Continue →** เพื่อไปยังขั้นตอนถัดไป
:::

---

## การจัดการพนักงาน

กดปุ่ม `...` ในคอลัมน์ Action หรือคลิกที่ชื่อพนักงานเพื่อดูรายละเอียดและแก้ไขข้อมูล
