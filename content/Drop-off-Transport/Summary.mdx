# Summary

หน้า **Summary** แสดงภาพรวมรายการ Drop-off Transport ที่มี Booking Confirmed ในแต่ละวัน เพื่อให้เจ้าหน้าที่สามารถจัดสรรยานพาหนะส่งลูกค้าได้จากหน้าเดียว

เข้าถึงได้จาก **Drop off Transport → Summary**

---

## ตารางรายการ

![Drop-off Transport Summary](/img/Drop-off-Transport/dropoff-summary.png)

ตารางแสดงรายการ Program ที่มี Booking Drop-off Transport ในวันที่เลือก ประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Program Name** | ชื่อโปรแกรม |
| **Destination** | จุดหมายปลายทาง |
| **Unit** | หน่วยและจำนวนผู้เดินทาง เช่น Person x 44 |
| **Total** | จำนวนผู้เดินทางรวม |
| **Action** | ปุ่ม `...` สำหรับจัดการ |

แถวสุดท้ายแสดง **Total** รวมจำนวนผู้เดินทางทั้งหมดของวันนั้น

:::info
หน้านี้แสดงเฉพาะ Program ที่มี Booking สถานะ **Confirmed** ในวันที่เลือกเท่านั้น
:::

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Date Filter** | เลือกวันที่เพื่อดูรายการ Drop-off Transport |
| **Search Program** | ค้นหา Program ตามชื่อ |
| **Manage Group** | เปิดหน้าจัดการกลุ่มแบบรวม |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** เพื่อเลือกวิธีจัดสรรยานพาหนะ

| ตัวเลือก | คำอธิบาย |
|---------|----------|
| **Supplier Group** | ไปยังหน้าจัดสรรยานพาหนะจาก Supplier ภายนอก |
| **Own Vehicle Group** | ไปยังหน้าจัดสรรยานพาหนะของบริษัทเอง |
