# Job Order

หน้า **Job Order** แสดงรายการใบสั่งงาน Drop-off Transport ที่สร้างขึ้นจากการจัดกลุ่ม สามารถดาวน์โหลดเพื่อส่งให้ Driver หรือ Supplier รับทราบจุดส่งลูกค้าและ Zone ที่รับผิดชอบ

เข้าถึงได้จาก **Drop off Transport → Job Order**

---

## ตารางรายการ

![Drop-off Job Order](/img/Drop-off-Transport/dropoff-job-order.png)

### แท็บสถานะ

| แท็บ | คำอธิบาย |
|------|----------|
| **All** | แสดง Job Order ทั้งหมด |
| **Pending** | Job Order ที่ยังไม่ดำเนินการ |
| **Completed** | Job Order ที่ดำเนินการเสร็จสิ้นแล้ว |

### คอลัมน์ตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **#** | ลำดับที่ |
| **Trip Code** | รหัส Trip |
| **Destination** | จุดหมายปลายทาง |
| **Zone** | Zone ที่รถรับผิดชอบ (เช่น Patong, Kata) |
| **Personnel** | บุคลากรที่มอบหมาย |
| **Registration** | ทะเบียนรถที่จัดสรร |
| **Status** | สถานะ Job Order |
| | <span class="badge badge-pending">Pending</span> รอดำเนินการ |
| | <span class="badge badge-completed">Completed</span> เสร็จสมบูรณ์ |
| **Action** | ปุ่มจัดการ |

:::info
Job Order จะถูกสร้างอัตโนมัติเมื่อมีการจัดสรร Booking เข้ากลุ่มใน **Supplier Group** หรือ **Own Vehicle Group** เสร็จสมบูรณ์
:::

---

## ฟังก์ชันบนหน้า

| ปุ่ม / ฟังก์ชัน | คำอธิบาย |
|----------------|----------|
| **Export** | ส่งออกรายการ Job Order (dropdown เลือกรูปแบบไฟล์) |
| **Date Filter** | เลือกวันที่ที่ต้องการดู Job Order |
| **Search** | ค้นหา Job Order |

---

## Action

กดปุ่ม `...` ในคอลัมน์ **Action** หรือกดที่ **Trip Code** เพื่อจัดการ Job Order

### View

แสดงรายละเอียด Job Order แบบ read-only พร้อมรายชื่อผู้โดยสาร, จุดส่ง และ Zone ที่รับผิดชอบ

### Download / Print

ดาวน์โหลดหรือพิมพ์ใบ Job Order เพื่อส่งให้ Driver หรือ Supplier
