Check In List

หน้า Check In List แสดงสรุปสถานะการ Check In ของลูกค้าในแต่ละวัน จัดกลุ่มตาม Trip พร้อมตัวเลข Waiting, Checked In และ No show แยกต่อ Trip

มี 2 ประเภท เข้าถึงได้จาก Check In → Excursion → Check In List และ Check In → Transport → Check In List

ตารางรายการ​

  • Excursion
  • Transport

Status Card​

ด้านบนแสดง 5 Card สรุปภาพรวมของวันที่เลือก

Cardคำอธิบาย
Waitingจำนวน Pax ที่ยังรอ Check In
Checked Inจำนวน Pax ที่ Check In แล้ว
No showจำนวน Pax ที่ไม่มาตามนัด
Amount Tripจำนวน Trip ทั้งหมดของวันนั้น
Amount Paxจำนวน Pax รวมทั้งหมด

แท็บสถานะ​

แท็บคำอธิบาย
Allแสดงทุก Trip
PendingTrip ที่ยังมี Waiting อยู่
CompletedTrip ที่ Waiting = 0

คอลัมน์ตาราง​

คอลัมน์คำอธิบาย
#ลำดับที่
Trip Codeรหัส Trip (คลิกได้เพื่อดูรายละเอียด)
Travel Dateวันที่เดินทาง
Trip Typeประเภท Trip (แสดง Join In badge ถ้าเป็น Join In)
Trip Roundรอบเวลาเดินทาง
Programชื่อโปรแกรม
Paxจำนวน Pax รวมของ Trip นี้
Waitingจำนวน Pax ที่ยังรอ (แสดงสีเหลือง)
Checked Inจำนวน Pax ที่ Check In แล้ว (แสดงสีเขียว)
No showจำนวน Pax ที่ไม่มาตามนัด (แสดงสีแดง)
Actionปุ่ม ... สำหรับจัดการ

แถวสุดท้ายแสดง Total รวม Pax, Waiting, Checked In, No show ทั้งหมด

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Date Filterเลือกวันที่ที่ต้องการดูรายการ
Searchค้นหา Trip หรือ Program
Exportส่งออกรายการ Check In เป็นไฟล์ (dropdown เลือกรูปแบบ)

Action​

กดปุ่ม ... หรือคลิก Trip Code เพื่อดูรายละเอียดภายใน Trip นั้น

View​

แสดงรายชื่อลูกค้าทั้งหมดภายใน Trip พร้อมสถานะ Waiting / Checked In / No show รายบุคคล สามารถเปลี่ยนสถานะหรือทำ Check In เพิ่มเติมได้จากหน้านี้
