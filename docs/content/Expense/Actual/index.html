Expenses — ค่าใช้จ่ายจริง

Expenses คือการบันทึกค่าใช้จ่ายที่เกิดขึ้นจริงระหว่างการดำเนินทริปหรือโปรแกรมทัวร์ หลังจากที่ได้รับ Advance แล้ว

ภาพรวม​

หน้า Expenses แสดงรายการค่าใช้จ่ายจริงทั้งหมด

คอลัมน์คำอธิบาย
#ลำดับที่
Trip / Programชื่อ Trip หรือโปรแกรม
Travel Dateวันที่เดินทาง
Item Typeประเภทค่าใช้จ่าย
Amountจำนวนเงินจริง
Advanceเงินที่เบิกล่วงหน้า
Differenceผลต่าง
Actionปุ่มจัดการ

การบันทึกค่าใช้จ่ายจริง​

  1. กดปุ่ม Create หรือ + Add
  2. เลือก Trip / Program ที่เกี่ยวข้อง
  3. เลือก Item Type ประเภทค่าใช้จ่าย
  4. กรอก Amount จำนวนเงินจริงที่จ่ายไป
  5. แนบหลักฐาน/ใบเสร็จ (ถ้ามี)
  6. กดบันทึก
tip

ระบบจะคำนวณผลต่างระหว่างเงิน Advance กับค่าใช้จ่ายจริงอัตโนมัติ หากจ่ายเกิน Advance ให้เบิกเพิ่ม หากเหลือให้คืนเงิน
