Program List

Program List คือรายละเอียด Product หรือรายละเอียด Program ของ Tour นั้นๆ

ตารางรายการ Program​

ตารางแสดงรายการ Program ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

คอลัมน์คำอธิบาย
#ลำดับที่
Program Nameชื่อ Program
Seller Quantityจำนวน Seller ที่ผูกกับ Program นั้นๆ
Statusสถานะการใช้งาน
Draft: ร่างข้อมูล ยังไม่สามารถใช้งานได้
Active เปิดใช้งาน
Inactive ปิดใช้งาน
Last Modify Dateวันที่สร้างหรือแก้ไขล่าสุด
Actionปุ่มจัดการสำหรับการ View / Edit
caution

ในโหมด View จะไม่สามารถแก้ไขข้อมูลได้ — หากต้องการแก้ไขให้กดปุ่ม Edit ภายในหน้า View

ฟังก์ชันบนหน้า​

ปุ่มคำอธิบาย
Createสร้าง Program ใหม่
Searchค้นหา Program ตามชื่อ
Filterกรองข้อมูลตามเงื่อนไข

ตัวกรอง​

กดปุ่ม Filter เพื่อเปิด Modal กรองข้อมูลตามเงื่อนไขที่ต้องการ

ฟิลด์คำอธิบาย
Program codeกรองตามรหัส Program
Program nameกรองตามชื่อ Program
Statusกรองตามสถานะ Draft / Active / Inactive

กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Create​

กดปุ่ม Create เพื่อสร้าง Program ของ Tour ใหม่ โดยมีข้อมูลที่ต้องกรอก 5 ส่วน ดังนี้

  • Program Detail
  • Itinerary
  • Cost
  • Unit
  • Seller

Program Detail​

กรอกข้อมูลเบื้องต้นของ Program

ฟิลด์คำอธิบาย
Program Nameชื่อของ Program tour นั้นๆ
Destinationจังหวัดของ Program นั้นๆ
Service Languageภาษาที่ให้บริการใน Program นั้นๆ
Package Availabilityวันที่เปิดขาย / วันเปิดให้บริการของ Program
Arrangement Typeประเภทการจัดโปรแกรม (ดูรายละเอียดด้านล่าง)
Booking Managementรูปแบบการยืนยัน Booking
Optionแพ็กเกจเสริมหรือโปรแกรมเสริมจากโปรแกรมหลัก (กรอกหรือไม่กรอกก็ได้) เช่น เก็บค่าธรรมเนียมอุทยาน หรือ รวมค่าธรรมเนียมอุทยาน

Arrangement Type​

เลือกประเภทการจัดโปรแกรม มี 3 ตัวเลือก

ประเภทคำอธิบายตัวอย่าง
Excursionกิจกรรมหรือโปรแกรมท่องเที่ยว (เป็นได้ทั้งเรือและรถ)Safari เป็นต้น
Transportจัดรถรับ-ส่งไปรับส่งสนามบิน-โรงแรม เป็นต้น
Excursion + Transportรถรับ-ส่งเพื่อไปทำกิจกรรม หลังจบกิจกรรมจะนำลูกค้ากลับจุดเดิมหรือจุดที่ลูกค้าแจ้งรับลูกค้าจากจุดหนึ่งไปอีกจุดเพื่อทำกิจกรรม เป็นต้น

Booking Management​

ตัวเลือกคำอธิบาย
Auto Confirmเมื่อสร้าง Booking จะ Confirm อัตโนมัติ
Manualเมื่อสร้าง Booking ต้องการกด Confirm ด้วยตนเอง

Itinerary​

กรอกรายละเอียด Trip ของ Program

ฟิลด์คำอธิบาย
Trip Roundรอบกิจกรรม / ช่วงเวลารอบ Trip นั้นๆ
Trip Typeประเภทการเดินทาง แบ่งเป็น Join In และ Private
Meeting Pointจุดนัดพบ เช่น ท่าเรือ หรือสถานที่ต่างๆ
Itineraryรายละเอียดโปรแกรมทัวร์
Previewตรวจสอบข้อมูลหลังจากกรอกเรียบร้อยแล้ว

Trip Type: Join In​

Join In คือประเภทการเข้าร่วมกับกลุ่มอื่นๆ (รวมกลุ่ม)

ฟิลด์คำอธิบาย
Minimum to runจำนวนลูกค้าขั้นต่ำสำหรับเริ่มต้นกิจกรรมนั้นๆ (กรอกหรือไม่กรอกก็ได้)
Meeting pointจุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร
Start Tripเวลาออกเดินทาง พร้อม Detail รายละเอียดเพิ่มเติม
Timeช่วงเวลาท่องเที่ยว พร้อม Detail และปุ่ม Add Time เพื่อเพิ่มช่วงเวลา
Drop off pointจุดส่งผู้โดยสาร ประกอบด้วย End Trip, Drop off point และปุ่ม Add Drop off

Trip Type: Private​

Private คือประเภทการ Trip แบบกลุ่มเดียวหรือส่วนตัว

ฟิลด์คำอธิบาย
Meeting pointจุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร
Start Tripเวลาออกเดินทาง พร้อม Detail รายละเอียดเพิ่มเติม
Timeช่วงเวลาท่องเที่ยว พร้อม Detail และปุ่ม Add Time เพื่อเพิ่มช่วงเวลา
Drop off pointจุดส่งผู้โดยสาร ประกอบด้วย End Trip, Drop off point และปุ่ม Add Drop off

Cost Details​

ค่าใช้จ่ายใน Program แบ่งออกเป็น 5 ส่วน ดังนี้

tip

หาก Itinerary เลือก Trip Type ทั้ง 2 ประเภท ส่วน Cost จะแสดงทั้ง 2 ประเภทด้วยเช่นกัน พร้อมปุ่ม Copy เพื่อคัดลอกข้อมูลจาก Trip Type หนึ่งไปยังอีกประเภท

Payment Option มี 2 ตัวเลือก

ตัวเลือกคำอธิบาย
Cashเงินสด
Creditเครดิต

Cost Type มี 3 ประเภท

ประเภทคำอธิบาย
Allรวมทั้งหมด (ราคาเหมา)
Personต่อคน
Fixคิดตามจำนวนที่ระบุ โดยมี Min (Pax) และ Max (Pax) ช่วงจำนวนคน (ผู้เดินทาง) ต่ำสุด–สูงสุด สำหรับเรทราคานั้นๆ

ตัวอย่าง Fix Cost
  • Min Pax = 1, Max Pax = 10, Cost = 100 → กลุ่ม 1–10 คน คิดราคา 100 บาท
  • กด Add Fix Cost เพื่อเพิ่มช่วงถัดไป → Min Pax = 11, Max Pax = 20, Cost = 200
caution
  • หากเลือก Included จะต้องกรอกข้อมูลให้ครบถ้วน หากไม่มีข้อมูลแนะนำให้เลือก Not Included
  • หากทำการเลือกข้อมูลมาแล้วจะต้องกรอกข้อมูลให้ครบถ้วน ไม่เช่นนั้นจะบันทึกไม่ได้
  • Guide Fee
  • Vehicle Cost

Guide Fee​

Guide Fee คือค่าจ้างไกด์ โดยมีตัวเลือก Included / Not Included

กดปุ่ม Add Guide เพื่อเลือกประเภท Guide ที่ต้องการ

Guide Itemคำอธิบาย
Guide 1ไกด์หลัก
Guide 2ไกด์รอง
Staffผู้ช่วย (สามารถระบุจำนวนผู้ช่วยสูงสุดในกิจกรรมนั้นๆ ได้)
ฟิลด์คำอธิบาย
Guide Typeเลือกประเภท Guide (Guide 1 / Guide 2 / Staff)
Payment OptionCash — เงินสด / Credit — เครดิต
Cost TypeAll / Person / Fix
Optionโปรแกรมเสริมที่ระบุใน Program Detail (หากไม่ระบุจะแสดง "–")
Cost (THB)ระบุค่าใช้จ่ายเป็นตัวเลขทศนิยม

Vehicle Cost​

Vehicle Cost คือค่าใช้จ่ายยานพาหนะ

กดปุ่ม Add Vehicle เพื่อเลือก Supplier หรือ Own Vehicle ที่มีการสร้างไว้

กรณีเลือก Supplier​

ระบบจะกรอง Supplier และ Group Type ตาม Arrangement Type ของ Program อัตโนมัติ

Arrangement TypeSupplier / Group Type ที่แสดง
ExcursionExcursion และ Excursion+Transport
TransportTransport และ Excursion+Transport
Excursion+TransportExcursion, Transport และ Excursion+Transport

เมื่อเลือก Supplier และ Group Type แล้วจะแสดงตามตัวอย่าง

กรณีเลือก Own Vehicle​

ระบบจะกรอง Group Type ตาม Arrangement Type เช่นเดียวกับ Supplier

เมื่อเลือก Group Type ของ Own Vehicle แล้วจะต้องเลือก Item Type เพื่อระบุว่าเป็นค่าอะไร เช่น ค่าน้ำมัน 91, ค่าที่จอดรถ เป็นต้น

ฟิลด์คำอธิบาย
Payment OptionCash — เงินสด / Credit — เครดิต
Supplier / Own Vehicleเลือก Supplier หรือ Own Vehicle
Vehicle (Group Type)ข้อมูลจาก Group Type ของ Vehicle
Optionโปรแกรมเสริมที่ระบุใน Program Detail (หากไม่ระบุจะแสดง "–")
Cost (THB)ระบุค่าใช้จ่ายเป็นตัวเลขทศนิยม

Other Expense​

Other Expense คือค่าใช้จ่ายอื่นๆ

กดปุ่ม Add Other Expense เพื่อเลือก Item Type ที่มีการสร้างไว้

หากยังไม่มี Item Type สามารถกด Create Item Type ได้เลย

หากมี Item Type แล้วแต่ยังไม่มี Sub Item ที่ต้องการ สามารถกด + Sub Items ได้

เมื่อเลือก Item Type แล้วจะแสดงตามตัวอย่าง

ฟิลด์คำอธิบาย
Payment OptionCash — เงินสด / Credit — เครดิต
Itemsตัวเลือกมาจาก Items Type ของ Program (รองรับ Sub Items)
Remarkหมายเหตุเกี่ยวกับ Items
Cost TypeAll / Person / Fix
Optionโปรแกรมเสริมที่ระบุใน Program Detail (หากไม่ระบุจะแสดง "–")
Cost (THB)ระบุค่าใช้จ่ายเป็นตัวเลขทศนิยม

Allowance​

Allowance คือค่าเบี้ยเลี้ยงพนักงาน

กดปุ่ม Add Allowance เพื่อเลือก Item Type ที่มีการสร้างไว้

เมื่อเลือก Item Type แล้วสามารถเลือก Position ที่ทำการสร้างไว้ได้ หากไม่มีที่ต้องการสามารถสร้างจาก Create Position ได้

Extra Cost​

Extra Cost คือค่าใช้จ่ายอื่นๆ เพิ่มเติมสำหรับแพ็กเกจเสริม (Option) นั้นๆ

เช่น รวมค่าธรรมเนียมอุทยาน และไม่รวมค่าธรรมเนียมอุทยาน เป็นต้น (ขึ้นอยู่กับดุลพินิจของคนสร้าง)

กดปุ่ม Add Other Item เพื่อเลือก Item Type ที่มีการสร้างไว้

เมื่อเลือกแล้วจะแสดงตามตัวอย่าง

Unit​

Unit คือตัวเลือกการจอง โดยค่าตั้งต้นที่มีมาให้คือ Infant (Default) ประเภท Infant (ประเภทนี้เป็นการเข้าหน้างานโดยไม่มีการจอง เช่น เด็กเล็ก)

กดปุ่ม Add Unit เพื่อเลือก Unit ตามที่มีการสร้างไว้ หากไม่มีสามารถกด Create New Unit ได้

Unit Name คือชื่อที่ใช้เรียกสำหรับตัวเลือกการจองนั้นๆ

Type มี 2 ประเภท

Typeคำอธิบาย
Personบุคคล — นับจำนวนคน
Optionตัวเลือกเสริม — ไม่นับจำนวนคน เช่น ค่าธรรมเนียมอุทยานเป็นตัวเลือก

info
  • หากเลือก Trip Type ทั้ง Join In และ Private ระบบจะแสดงฟิลด์ Unit ทั้ง 2 ประเภท

Seller​

Seller คือตัวแทนขาย

กดปุ่ม Add Seller เพื่อเลือก Seller ตามที่มีการสร้างไว้ หากไม่มีสามารถกด Create Seller ได้

ฟิลด์คำอธิบาย
Seller Nameชื่อตัวแทนขาย
Booking No.หากเลือก Auto Booking No. ระบบจะสร้างหมายเลข Booking ให้อัตโนมัติ (สามารถแก้ไขได้) หากไม่เลือกต้องกรอกหมายเลขเอง
Time Date Detailช่วงวันที่ให้บริการ (From / End)
Unit Detail & Selling Priceกำหนดราคาขายแยกตาม Unit

เมื่อเลือก Seller แล้วจะแสดงตามตัวอย่าง (หากยังไม่ต้องการกรอกข้อมูลสามารถกดย่อที่ลูกศรได้)

info

หากยังไม่มีการสร้าง Seller ไว้ในระบบ จะไม่มีข้อมูลให้เลือก ยังไม่ต้องใส่ในขั้นตอนนี้ก็ได้
