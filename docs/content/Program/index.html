Program List

Program List รายละเอียด Product หรือรายละเอียด Program ของ Tour นั้นๆ โดยมีรายละเอียดสำคัญดังนี้

  • ปุ่ม Create สำหรับสร้าง Program ของ Tour
  • Agent Quantity แสดงจำนวน Agent ของ Program นั้นๆ
  • Status แสดงสถานะ เปิด-ปิด การใช้งาน Program
  • Last Modify Date แสดงวันที่ สร้างหรือแก้ไขล่าสุด
  • Action สำหรับจัดการโปรแกรมทัวร์

![รูป]

Create​

เมื่อกด Create จะเข้าสู่หน้าสร้าง Program ของ Tour โดยมีข้อมูลที่ต้องกรอกทั้งหมด 5 ส่วนใหญ่ๆ ดังนี้

  • Program Detail
  • Itinerary
  • Cost
  • Unit
  • Agent

Program Detail​

หน้าต่างนี้กรอกข้อมูลเบื้องต้นของ Program

  • Program Name ชื่อของ Program นั้นๆ
  • Province จังหวัดของ Program นั้นๆ
  • Language ภาษาที่ใช้ใน Program นั้นๆ
  • Package Availability วันเปิดให้บริการของ Program
  • Option หมายถึง ช่องทางเสริมหรือโปรแกรมเสริมจากโปรแกรมหลัก เป็นส่วนที่กรอกหรือไม่กรอกก็ได้ เช่น โปรแกรม 1 มี Option เป็น บวกอาหารกลางวัน เป็นต้น

![รูป]

Itinerary​

หน้าต่างนี้กรอกเกี่ยวกับข้อมูลรายละเอียด Trip ของ Program

  • Trip Round กำหนดช่วงเวลารอบ Trip นั้นๆ
  • Trip Type แบ่งออกเป็น 2 ประเภท
    • Join In คือ ประเภทการเข้าร่วมกับกลุ่มอื่นๆ โดยเมื่อกดเลือก Join In แล้ว จะแสดง Minimum to run หากต้องการกำหนดจำนวนคนที่ต่ำที่สุดที่จะออกเดินทาง และมีรายละเอียดที่ต้องกรอกข้อมูลเพิ่มเติม ดังนี้ ![รูป]
      • Meet up point / Pick up point โดยมีให้เลือกอย่างใดอย่างหนึ่ง และกรอกเพื่อกำหนดสถานที่นัดพบ
        • Meet up point หมายถึงจุดรวมพลหรือรวมกลุ่มเพื่อรับผู้โดยสาร
        • Pick up point หมายถึงจุดนัดรับที่จะเข้ารับผู้โดยสาร
      • Start Trip หมายถึง เวลาในการออกเดินทาง
        • Detail รายละเอียดใน Start Trip
      • Time หมายถึง ช่วงเวลาที่เดินทางถึงสถานที่ท่องเที่ยว/ช่วงเวลาท่องเที่ยว
        • Detail รายละเอียดในการท่องเที่ยวของช่วงเวลาที่กำหนด
        • Add Time เพื่อใช้ในการเพิ่มช่วงเวลาท่องเที่ยว
      • Drop off point กำหนดช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
        • End Trip ช่วงเวลาที่ถึงสถานที่จัดส่งผู้โดยสาร
        • Drop off point สถานที่ที่จัดส่งผู้โดยสาร
        • Add Drop off เพื่อใช้ในการเพิ่มช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
    • Private คือ ประเภทการ Trip แบบกลุ่มเดียวหรือ ส่วนตัว มีรายละเอียดที่ต้องกรอกข้อมูลเพิ่มเติม ดังนี้ ![รูป]
      • Start Trip หมายถึง เวลาในการออกเดินทาง
        • Detail รายละเอียดใน Start Trip
      • Time หมายถึง ช่วงเวลาที่เดินทางถึงสถานที่ท่องเที่ยว/ช่วงเวลาท่องเที่ยว
        • Detail รายละเอียดในการท่องเที่ยวของช่วงเวลาที่กำหนด
        • Add Time เพื่อใช้ในการเพิ่มช่วงเวลาท่องเที่ยว
      • Drop off point กำหนดช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสาร
        • End Trip ช่วงเวลาที่ถึงสถานที่จัดส่งผู้โดยสาร
        • Drop off point สถานที่ที่จัดส่งผู้โดยสาร
        • Add Drop off เพื่อใช้ในการเพิ่มช่วงเวลาและสถานที่ที่จัดส่งผู้โดยสา

Cost​

เกี่ยวกับค่าใช้จ่ายใน Program โดยแบ่งออกเป็น 3 ส่วน ดังนี้ (หาก Itinerary มีการเลือก Trip Type 2 ประเภท จะปรากฏในหน้านี้ 2 ประเภทด้วยเช่นกัน โดยมีปุ่ม Copy เพื่อให้สะดวกสิ่งที่กรอกไปยัง Trip Type หนึ่ง)

  • Guide
  • Vehicle
  • Other Expense
  • Guide ค่าใช้จ่ายเกี่ยวกับ Guide
    • Check Box : Cash Advance หากติ๊กจะแสดงให้รู้ว่าต้องการให้จ่ายก่อน-หากไม่ติ๊กจ่ายหลัง
    • Guide Item ให้เลือกระหว่างเป็น Guide หรือ Staff
    • Cost Type มีทั้งหมด 3 ประเภท ดังนี้
      • All คือ ราคาทั้งหมดกี่บาท (แบบเหมา)
      • Person คือ ราคาต่อคนกี่บาท
      • Fix คือ จะมาพร้อมกับจำนวนคนเพื่อระบุว่า จำนวนคน(น้อยสุด) / จำนวนคน(มากสุด) ราคากี่บาท
    • Option คือ ช่องทางเสริม/โปรแกรมเสริมที่ระบุไว้ในขั้นตอน Program Detail หากไม่ได้ระบุจะไม่แสดง "-"
    • Cost(THB) ระบุค่าใ้ช้จ่าย กำหนดเป็นตัวเลข จุดทศนิยม
    • Add Guide Cost ไว้สำหรับเพิ่มจำนวนค่าใช้จ่าย Guide
  • Vehicle ค่าใช้จ่ายเกี่ยวกับ Vehicle
    • Check Box : Cash Advance หากติ๊กจะแสดงให้รู้ว่าต้องการให้จ่ายก่อน-หากไม่ติ๊กจ่ายหลัง
    • Supplier ข้อมูลมาจาก Supplier หากไม่่ได้เพิ่ม Supplier ไว้จะไม่แสดงข้อมูล (สามารถมาใส่ทีหลังได้)
    • Vehicle ข้อมูลมาจาก Group Type ของ Vehicle
    • Option คือ ช่องทางเสริม/โปรแกรมเสริมที่ระบุไว้ในขั้นตอน Program Detail หากไม่ได้ระบุจะไม่แสดง "-"
    • Cost(THB) ระบุค่าใ้ช้จ่าย กำหนดเป็นตัวเลข จุดทศนิยม
    • Add Vehicle ไว้สำหรับเพิ่มจำนวนค่าใช้จ่าย Vehicle
  • Other Expense ค่าใช้จ่ายอื่นๆ
    • Check Box : Cash Advance หากติ๊กจะแสดงให้รู้ว่าต้องการให้จ่ายก่อน-หากไม่ติ๊กจ่ายหลัง
    • Items ข้อมูลตัวเลือกมาจาก Items Type ของ Program
    • Remark กรอกข้อมูลที่ต้องการแจ้งได้เกี่ยวกับ Items
    • Cost Type มีทั้งหมด 3 ประเภท ดังนี้
      • All คือ ราคาทั้งหมดกี่บาท (แบบเหมา)
      • Person คือ ราคาต่อคนกี่บาท
      • Fix คือ จะมาพร้อมกับจำนวนคนเพื่อระบุว่า จำนวนคน(น้อยสุด) / จำนวนคน(มากสุด) ราคากี่บาท
    • Option คือ ช่องทางเสริม/โปรแกรมเสริมที่ระบุไว้ในขั้นตอน Program Detail หากไม่ได้ระบุจะไม่แสดง "-"
    • Cost(THB) ระบุค่าใ้ช้จ่าย กำหนดเป็นตัวเลข จุดทศนิยม
    • Add Other Expense ไว้สำหรับเพิ่มจำนวนค่าใช้จ่าย Guide

Unit​

Unit เป็นหน่วยการจอง โดยค่าตั้งต้นที่มีมาให้คือ ชื่อ Infant ประเภท Infant (ประเภทนี้เป็นการเข้าหน้างานไม่ได้มีการจอง เช่น เด็กเล็ก เป็นต้น) โดยจะแบ่งเป็น Trip Type หากเลือกเข้ามาเป็น Join In และ Private ระบบจะขึ้นมาให้กรอกสองประเภท

โดยสามารถเลือก Create Unit ในหน้านี้ได้เลย

สามารถ Add Unit เพิ่มได้ เพื่อระบุ เช่น ขายเป็นกลุ่ม, ขายเดี่ยว หรือขายเด็ก

Agent​

Agent เป็นการระบุว่า Agent ใดใช้โปรแกรมนี้บ้าง (ยังไม่ต้องใส่ได้เช่นกัน) หากไม่มีการสร้าง Agent ก่อนจะไม่มีข้อมูลปรากฏขึ้น

Agent Quantity​

แสดงจำนวนของ Agent ที่มีการใช้งาน เมื่อนำเม้าส์ชี้ จะแสดงรายชื่อ Agent ที่มีการผูกกับ Program นั้นๆ

Status​

แสดงบอกสถานะ เปิด-ปิด การใช้งานสิทธิการเข้าถึงนั้นๆ โดยแบ่งออกเป็น 2 สถานะดังนี้

  • Active บ่งบอกถึงสถานะนี้ เปิดการใช้งาน สิทธิการเข้าถึง แสดงสีเป็นสีเขียว
  • Inactivate บ่งบอกถึงสถานะนี้ ปิดการใช้งาน สิทธิการเข้าถึง แสดงสีเป็นสีแดง

Last Modify Date​

แสดงวันที่ที่ทำการสร้างหรือแก้ไขข้อมูลล่าสุด

Action​

สามารถจัดการข้อมูลด้วยคำสั่งต่างๆ ได้ดังรูป

  • View เพื่อเข้าตรวจสอบข้อมูลที่มีการสร้างขึ้นหรือแก้ไขล่าสุดแต่จะ ไม่สามารถแก้ไขข้อมูล หากต้องการแก้ไขข้อมูลสามารถกด Edit ได้

  • Delete เพื่อใช้ในการลบข้อมูล Program ที่ไม่ได้ใช้งานแล้วหรือต้องการลบออกจากระบบ
