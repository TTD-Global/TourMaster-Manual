Report — Excursion

Report ในส่วน Excursion คือรายงานสรุปข้อมูล Excursion ตามช่วงเวลาที่เลือก

การดูรายงาน​

  1. เลือกช่วงวันที่จาก Date Filter
  2. ระบบจะสร้างรายงานสรุปข้อมูล Excursion อัตโนมัติ
  3. กดปุ่ม Export เพื่อดาวน์โหลดรายงานเป็นไฟล์ Excel

ข้อมูลในรายงาน​

รายงานจะแสดงข้อมูลเช่น

  • สรุปจำนวน Excursion ตามโปรแกรม
  • ข้อมูล Supplier / Own Vehicle ที่ใช้งาน
  • จำนวน Pax รวม
  • ยอดรวมค่าใช้จ่าย
