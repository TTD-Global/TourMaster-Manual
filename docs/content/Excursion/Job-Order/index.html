Job Order

หน้า Job Order แสดงรายการใบสั่งงาน Excursion ที่สร้างขึ้นจากการจัดกลุ่ม สามารถดาวน์โหลดเพื่อส่งให้ Driver หรือ Supplier รับทราบงานได้

เข้าถึงได้จาก Excursion → Job Order

ตารางรายการ​

แท็บสถานะ​

แท็บคำอธิบาย
Allแสดง Job Order ทั้งหมด
PendingJob Order ที่ยังไม่ดำเนินการ
CompletedJob Order ที่ดำเนินการเสร็จสิ้นแล้ว

คอลัมน์ตาราง​

คอลัมน์คำอธิบาย
#ลำดับที่
Trip Codeรหัส Trip
Travel Dateวันที่เดินทาง
Trip Typeประเภท Trip
Trip Roundรอบเวลาเดินทาง
Programชื่อโปรแกรม Excursion
Paxจำนวนผู้เดินทาง
Registrationทะเบียนรถที่จัดสรร
Statusสถานะ Job Order (Pending / Completed)
Actionปุ่มจัดการ
info

Job Order จะถูกสร้างอัตโนมัติเมื่อมีการจัดสรร Booking เข้ากลุ่มใน Supplier Group หรือ Own Vehicle Group เสร็จสมบูรณ์

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Exportส่งออกรายการ Job Order (dropdown เลือกรูปแบบไฟล์)
Date Filterเลือกวันที่ที่ต้องการดู Job Order
Searchค้นหา Job Order
Advance Searchค้นหาขั้นสูงตามเงื่อนไขหลายอย่าง

ตัวกรอง​

Date Filter​

เลือกวันที่ที่ต้องการดู Job Order โดย default แสดงข้อมูลวันปัจจุบัน

Search​

พิมพ์ Trip Code หรือ Program เพื่อค้นหารายการ

Advance Search​

ค้นหาขั้นสูงโดยกรองจากหลายเงื่อนไขพร้อมกัน เช่น Trip Type, Program, Status กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

กดปุ่มในคอลัมน์ Action เพื่อจัดการ Job Order

View​

แสดงรายละเอียด Job Order แบบ read-only พร้อมรายชื่อผู้โดยสารและข้อมูลรถ

Download / Print​

ดาวน์โหลดหรือพิมพ์ใบ Job Order เพื่อส่งให้ Driver หรือ Supplier
