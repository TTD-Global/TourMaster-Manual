Supplier

Supplier List รายชื่อผู้ให้บริการยานพาหนะ โดยมีรายละเอียดสำคัญดังนี้

  • ปุ่ม Create สำหรับสร้างรายชื่อผู้ให้บริการยานพาหนะ
  • Group Type Quantity แสดงจำนวนประเภทยานพาหนะนั้นๆ
  • Status แสดงสถานะ เปิด-ปิด การใช้งานผู้ให้บริการยานพาหนะ
  • Last Modify Date แสดงวันที่ สร้างหรือแก้ไขล่าสุด
  • Action สำหรับจัดการรายชื่อผู้ให้บริการยานพาหนะ

Create​

เมื่อกด Create จะเข้าสู่หน้าสร้างรายชื่อผู้ให้บริการยานพาหนะ โดยมีข้อมูลที่ต้องกรอกทั้งหมด 2 ส่วน ดังนี้

  • General Detail
  • Supplier login

General Detail​

ข้อมูลทั่วไปของ Supplier มีทั้งหมด 3 ส่วนดังนี้

  • General Detail
  • Bank Detail
  • Group Type Detail
  • General Detail ข้อมูลทั่วไปของ Supplier มีดังนี้
    • Supplier Name ชื่อของ Supplier
    • Contract Person ชื่อผู้ทำสัญญา
    • Line ID ชื่อบัญชี Line
    • Phone Number หมายเลขเบอร์โทรศัพท์
    • Email Email ของ Supplier
    • Address ที่อยู่
    • Country ประเทศ
    • Province จังหวัด
    • District ตำบล/แขวง
    • Sub-district อำเภอ/เขต
    • Postal Code หมายเลขไปรษณีย์
  • Bank Detail ข้อมูลของธนาคาร
    • Payment Method โดยจะมีให้เลือกทั้งหมด 4 แบบ ดังนี้
      • Bank Transfer โอนเงิน โดยข้อมูลจะต้องเลือก ธนาคาร กรอกเลขบัญชีธนาคาร และชื่อผู้ถือบัญชี
      • Cash เงินสด
      • Credit Card บัตรเครดิต โดยจะต้องเลือกประเภทบัตรเครดิต และกรอกชื่อผู้ถือบัตรเครดิต
      • Cheque เช็คธนาคาร
  • Group Type Detail ข้อมูลประเภทรถที่จะใช้
    • Group Type สามารถเลือกข้อมูลได้มากกว่า 1 ตัวเลือก หากเลือก Select All ระบบจะดำเนินการเลือกตัวเลือกที่มีอยู่ทั้งหมด
    • Add Group Type ระบบแสดง Modal : Create Group Type แบ่งเป็นแสดงข้อมูล Group Type ที่มีอยู่ และ สร้าง
  • Supplier login สร้างบัญชีที่จะใช้ Login : Driver เพื่อตรวจสอบ จัดการ และสร้างคนขับ
    • Username ชื่อบัญชีเข้าสู่ระบบ
    • Password รหัสผ่านในการเข้าสู่ระบบ
    • Confirm Password ยืนยันรหัสผ่านในการเข้าสู่ระบบ

Group Type Quantity​

แสดงจำนวนของยานพาหนะที่มีการเลือกผูกไว้ตอนสร้างหรือแก้ไขข้อมูลล่าสุด

Last Modify Date​

แสดงวันที่ที่มีการสร้างหรือแก้ไขข้อมูลล่าสุด

Action​

สามารถจัดการข้อมูลด้วยคำสั่งต่างๆ ได้ดังรูป

  • View เพื่อเข้าตรวจสอบข้อมูลที่มีการสร้างขึ้นหรือแก้ไขล่าสุดแต่จะ ไม่สามารถแก้ไขข้อมูล หากต้องการแก้ไขข้อมูลสามารถกด Edit ได้

    • Edit ใช้ในการแก้ไขข้อมูลต่างๆรวมถึง สถานะ เปิด-ปิดการใช้งาน และสามารถ Reset Password ได้

  • Delete เพื่อใช้ในการลบข้อมูล Supplier ที่ไม่ได้ใช้งานแล้วหรือต้องการลบออกจากระบบ
