Guide / Staff

หน้า Guide / Staff ใช้จัดการรายชื่อพนักงานประเภท Guide และ Staff ภายในองค์กร

เข้าถึงได้จาก Setting → Employee → Guide / Staff

ตารางรายการ​

หน้านี้แบ่งเป็น 2 แท็บตามประเภทพนักงาน

  • Guide List
  • Staff List

Guide List​

ตารางแสดงรายการพนักงาน Guide ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

คอลัมน์คำอธิบาย
#ลำดับที่
Imageรูปโปรไฟล์พนักงาน
Nicknameชื่อเล่น / รหัสย่อ
Guide Nameชื่อ-นามสกุล Guide (คลิกได้เพื่อดูรายละเอียด)
Phone Numberเบอร์โทรศัพท์
License No.หมายเลขใบอนุญาต
Work Areaจำนวนพื้นที่ทำงานที่มอบหมาย
Languageจำนวนภาษาที่ใช้งานได้
Statusสถานะการใช้งาน
Active เปิดใช้งาน
Inactive ปิดใช้งาน
Draft ร่างข้อมูล ยังไม่เผยแพร่
Last Modify Dateวันที่แก้ไขล่าสุด
Actionปุ่ม ... สำหรับจัดการ

ฟังก์ชันบนหน้า​

ปุ่มคำอธิบาย
Create Guide / Create Staffสร้างพนักงานใหม่ (ปุ่มเปลี่ยนตามแท็บที่เลือก)
Searchค้นหาพนักงาน
Filterกรองข้อมูลตามเงื่อนไข
Exportดาวน์โหลดรายชื่อพนักงาน

ตัวกรอง​

กดปุ่ม Filter เพื่อเปิด Modal กรองข้อมูลตามเงื่อนไขที่ต้องการ

ฟิลด์คำอธิบาย
Nameกรองตามชื่อพนักงาน
Nick Nameกรองตามชื่อเล่น
Statusกรองตามสถานะ Active / Inactive

กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

กดปุ่ม ... ในคอลัมน์ Action เพื่อเลือกคำสั่ง หรือคลิกที่ชื่อพนักงานเพื่อเข้าดูข้อมูล

ข้อควรระวัง

ในโหมด View จะไม่สามารถแก้ไขข้อมูลได้ — หากต้องการแก้ไขให้กดปุ่ม Edit ภายในหน้า View

View​

แสดงข้อมูลพนักงานทั้งหมดแบบ read-only ตามประเภท (Guide แสดงครบ 3 ส่วน / Staff แสดง 2 ส่วน)

  • กดปุ่ม Edit ภายในหน้า View เพื่อเข้าสู่โหมดแก้ไข

Edit​

แก้ไขข้อมูลพนักงานได้ทุกส่วน เช่นเดียวกับหน้า Create

  1. แก้ไขข้อมูลในฟิลด์ที่ต้องการ
  2. กด Save & Approve เพื่อบันทึกการเปลี่ยนแปลง หรือ Cancel เพื่อยกเลิก

Create​

  • Guide List
  • Staff List

Create Guide​

กดปุ่ม Create Guide แล้วกรอกข้อมูลตาม 3 ขั้นตอน

  • Step 1 — General Detail
  • Step 2 — Guide Detail
  • Step 3 — Bank Detail

General Detail​

กรอกข้อมูลพื้นฐานของพนักงาน

ฟิลด์คำอธิบายจำเป็น
รูปโปรไฟล์อัปโหลดรูปพนักงาน (JPG / PNG / HEIC ไม่เกิน 5 MB)—
Titleคำนำหน้าชื่อ (Mr. / Ms. / Mrs.)—
First Nameชื่อ✅
Last Nameนามสกุล—
Nicknameชื่อเล่น / รหัสย่อสำหรับใช้ในระบบ✅
ID Card Numberหมายเลขบัตรประชาชน 13 หลัก✅
Date of Birthวันเกิด—
Phone Numberเบอร์โทรศัพท์✅
Line IDไลน์ไอดี—
Line User IDLine User ID (สำหรับ integration ระบบ)—
Upload ID Card Number Photosรูปถ่ายบัตรประชาชน (JPG / PNG / HEIC / JPEG ไม่เกิน 5 MB)—

กด Continue → เพื่อไปยังขั้นตอนถัดไป หรือ Save and exit เพื่อบันทึกแล้วกลับมาแก้ไขภายหลัง

Guide Detail​

กรอกข้อมูลสำหรับการมอบหมายงาน

ฟิลด์คำอธิบายจำเป็น
Guide Licenceหมายเลขใบอนุญาต Guide✅
License Expire Dateวันหมดอายุใบอนุญาต—
Languageภาษาที่ใช้งานได้ เช่น Thai, English, Chinese✅
Work Areaพื้นที่ทำงาน เช่น Bangkok, Phuket✅
Upload License Photosรูปถ่ายใบอนุญาต (JPG / PNG / HEIC / JPEG ไม่เกิน 5 MB)—

กด Continue → เพื่อไปยังขั้นตอนถัดไป หรือ Save and exit เพื่อบันทึกแล้วกลับมาแก้ไขภายหลัง

ฟิลด์คำอธิบายจำเป็น
Bank Accountเลือกธนาคาร (dropdown)—
Account Nameชื่อบัญชีธนาคาร—
Account Numberหมายเลขบัญชีธนาคาร—
Upload Bank passbook pageรูปถ่ายหน้าสมุดบัญชี—

กด Save & Approve เพื่อบันทึกข้อมูลทั้งหมดและสร้าง Guide

เคล็ดลับ

สามารถกด Save and exit ในแต่ละขั้นตอนเพื่อบันทึกก่อนแล้วกลับมากรอกต่อภายหลังได้
ไม่จำเป็นต้องกรอกให้ครบทุกขั้นตอนในครั้งเดียว
