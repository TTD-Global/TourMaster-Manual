Guide / Staff

หน้า Guide / Staff ใช้จัดการรายชื่อพนักงานประเภท Guide และ Staff ภายในองค์กร

เข้าถึงได้จาก Setting → Employee → Guide / Staff

ภาพรวม​

หน้านี้แบ่งเป็น 2 แท็บ

แท็บคำอธิบาย
Guide Listรายการพนักงาน Guide ทั้งหมด
Staff Listรายการพนักงาน Staff ทั้งหมด

คอลัมน์ในตาราง​

คอลัมน์คำอธิบาย
#ลำดับที่
Imageรูปโปรไฟล์พนักงาน
Nicknameชื่อเล่น / รหัสย่อ
Guide Name / Staff Nameชื่อ-นามสกุลพนักงาน (คลิกได้เพื่อดูรายละเอียด)
Phone Numberเบอร์โทรศัพท์
License No.หมายเลขใบอนุญาต
Work Areaจำนวนพื้นที่ทำงานที่มอบหมาย
Languageจำนวนภาษาที่ใช้งานได้
Statusสถานะ: Active (เขียว) / Draft (เทา)
Last Modify Dateวันที่แก้ไขล่าสุด
Actionปุ่ม ... สำหรับจัดการ

ฟังก์ชันบนหน้า​

ปุ่มคำอธิบาย
Create Guide / Create Staffสร้างพนักงานใหม่ (ปุ่มเปลี่ยนตามแท็บที่เลือก)
Searchค้นหาพนักงาน
Filterกรองข้อมูลตามเงื่อนไข
Exportดาวน์โหลดรายชื่อพนักงาน

การสร้าง Guide / Staff ใหม่​

กดปุ่ม Create Guide หรือ Create Staff แล้วทำตาม 3 ขั้นตอน

ขั้นตอนที่ 1 — General Detail​

กรอกข้อมูลทั่วไปของพนักงาน

ฟิลด์คำอธิบายจำเป็น
รูปโปรไฟล์อัปโหลดรูปพนักงาน—
Titleคำนำหน้าชื่อ✅
First Nameชื่อ✅
Last Nameนามสกุล✅
Nicknameชื่อเล่น / รหัสย่อ✅
ID Card Numberหมายเลขบัตรประชาชน✅
Date of Birthวันเกิด—
Line IDไลน์ไอดี—
Phone Numberเบอร์โทรศัพท์✅
Line User IDLine User ID (สำหรับ integration)—
Upload ID Card Number Photosรูปถ่ายบัตรประชาชน (JPG/PNG/HEIC/JPEG ไม่เกิน 5 MB)✅

ขั้นตอนที่ 2 — Guide Detail​

กรอกข้อมูลสำหรับการมอบหมายงาน Guide เช่น ภาษา, พื้นที่ทำงาน, ใบอนุญาต

ขั้นตอนที่ 3 — Bank Detail​

กรอกข้อมูลบัญชีธนาคารสำหรับการจ่ายค่าตอบแทน

tip

กดปุ่ม Save and exit เพื่อบันทึกข้อมูลในขั้นตอนปัจจุบันก่อน แล้วกลับมาแก้ไขต่อภายหลังได้ กดปุ่ม Continue → เพื่อไปยังขั้นตอนถัดไป

การจัดการพนักงาน​

กดปุ่ม ... ในคอลัมน์ Action หรือคลิกที่ชื่อพนักงานเพื่อดูรายละเอียดและแก้ไขข้อมูล
