Other

หน้า Other ใช้จัดการรายชื่อพนักงานประเภทอื่นๆ ที่ไม่ใช่ Guide หรือ Staff เช่น Driver, IT Support หรือตำแหน่งที่สร้างขึ้นเพิ่มเติมเอง

เข้าถึงได้จาก Setting → Employee → Other

ตารางรายการ​

ตารางแสดงรายการพนักงาน Other ทั้งหมด ประกอบด้วยคอลัมน์ดังนี้

คอลัมน์คำอธิบาย
#ลำดับที่
Imageรูปโปรไฟล์พนักงาน
Positionตำแหน่งงาน
Nameชื่อ-นามสกุลพนักงาน
Phone Numberเบอร์โทรศัพท์
Registrationยานพาหนะที่มอบหมาย
Statusสถานะการใช้งาน
Active เปิดใช้งาน
Inactive ปิดใช้งาน
Last Modify Dateวันที่แก้ไขล่าสุด
Actionปุ่ม ... สำหรับจัดการ

ฟังก์ชันบนหน้า​

ปุ่มคำอธิบาย
Create Otherสร้างพนักงานประเภท Other ใหม่
Searchค้นหาพนักงาน
Filterกรองข้อมูลตามเงื่อนไข
Exportดาวน์โหลดรายชื่อพนักงาน

ตัวกรอง​

กดปุ่ม Filter เพื่อเปิด Modal กรองข้อมูลตามเงื่อนไขที่ต้องการ

ฟิลด์คำอธิบาย
Positionsกรองตามตำแหน่ง
Nameกรองตามชื่อพนักงาน
Statusกรองตามสถานะ Active / Inactive

กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

กดปุ่ม ... ในคอลัมน์ Action เพื่อเลือกคำสั่ง

View​

แสดงข้อมูลพนักงานทั้งหมดแบบ read-only

  • กดปุ่ม Edit ภายในหน้า View เพื่อเข้าสู่โหมดแก้ไข

Edit​

  • แก้ไขข้อมูลได้ทุกฟิลด์
  • Status — เปลี่ยนสถานะระหว่าง Active และ Inactive
  • กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก
ข้อควรระวัง

ในโหมด View จะไม่สามารถแก้ไขข้อมูลได้ — หากต้องการแก้ไขให้กดปุ่ม Edit ภายในหน้า View

Create​

กดปุ่ม Create Other แล้วกรอกข้อมูลในฟอร์ม

ฟิลด์คำอธิบายจำเป็น
Photoอัปโหลดรูปพนักงาน—
Positionตำแหน่งงาน (เลือกจาก Positions ที่สร้างไว้)✅
Nameชื่อ-นามสกุลพนักงาน✅
Phone Numberเบอร์โทรศัพท์✅
Emailอีเมลพนักงาน—
Registrationยานพาหนะที่มอบหมาย—
  1. กรอกข้อมูลในฟอร์ม
  2. กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก
info

ตำแหน่งงาน (Position) ที่ใช้ในหน้า Other จะดึงมาจากข้อมูลที่สร้างไว้ใน Employee → Positions
ควรสร้าง Position ก่อนเพิ่มข้อมูลพนักงาน
