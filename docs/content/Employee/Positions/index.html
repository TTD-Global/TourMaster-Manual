Positions — ตำแหน่งงาน

Positions คือการตั้งค่าตำแหน่งงานต่างๆ ที่ใช้ในระบบ เช่น Tour Guide, Driver, Staff, Manager ฯลฯ ข้อมูลเหล่านี้จะนำไปใช้ตอนสร้างพนักงาน

ภาพรวม​

ตาราง Positions แสดงรายการตำแหน่งงานทั้งหมด

คอลัมน์คำอธิบาย
#ลำดับที่
Position Nameชื่อตำแหน่งงาน
Statusสถานะ (Active / Inactive)
Actionปุ่มจัดการ (Edit / Delete)

การสร้าง Position ใหม่​

  1. กดปุ่ม Create
  2. กรอก Position Name ชื่อตำแหน่งงาน
  3. กดปุ่ม Save

การจัดการ Position​

  • Edit — แก้ไขชื่อตำแหน่ง
  • Delete — ลบตำแหน่ง (ไม่สามารถลบได้หากมีพนักงานใช้ตำแหน่งนั้นอยู่)
caution

ควรสร้าง Position ก่อนเพิ่มข้อมูลพนักงาน เนื่องจากฟิลด์ Position เป็นฟิลด์บังคับ
