Booking List

Booking List คือหน้าสำหรับดูและจัดการรายการ Booking ทั้งหมดในระบบ

ตารางรายการ​

Status Card​

ด้านบนตารางแสดงจำนวน Booking แยกตามกลุ่มสถานะ สามารถกดที่ Card เพื่อกรองตารางตามสถานะนั้นได้ทันที

In Progress — Booking ที่อยู่ระหว่างดำเนินการ

สถานะคำอธิบาย
ReservedBooking ที่สร้างแล้ว รอการยืนยัน
ConfirmedBooking ที่ได้รับการยืนยันแล้ว

Processed — Booking ที่ดำเนินการเสร็จสิ้น (กรองตามเดือนปัจจุบัน)

สถานะคำอธิบาย
CompletedBooking ที่เสร็จสมบูรณ์
CanceledBooking ที่ถูกยกเลิก

คอลัมน์ตาราง​

คอลัมน์คำอธิบาย
#ลำดับที่
Sellerชื่อ Seller หรือตัวแทนที่รับจอง
Travel Dateวันที่เดินทาง
Booking Dateวันที่สร้าง Booking
Booking No.เลขที่ Booking (คลิกได้เพื่อดูรายละเอียด)
Trip Typeประเภทของ Trip
Travel Timeเวลาเดินทาง
Programชื่อโปรแกรมทัวร์
Paxจำนวนผู้เดินทาง
Trip Codeรหัส Trip
Statusสถานะปัจจุบัน
Actionปุ่มจัดการ

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Create Bookingสร้าง Booking ใหม่
Date Filterเลือกวันที่ที่ต้องการดู Booking
Advance Searchค้นหาขั้นสูง กรองตามเงื่อนไขหลายอย่างพร้อมกัน
Exportส่งออกรายการ Booking เป็นไฟล์ Excel

ตัวกรอง​

Date Filter​

เลือกวันที่ที่ต้องการดูรายการ Booking สามารถสลับโหมดได้ระหว่าง Booking Date (วันที่สร้าง Booking) และ Travel Date (วันที่เดินทาง)

Advance Search​

ค้นหาขั้นสูงโดยกรองจากหลายเงื่อนไขพร้อมกัน เช่น Seller, Program, Trip Type, Status กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

คลิกที่ Booking No. หรือกดปุ่มในคอลัมน์ Action เพื่อจัดการ Booking

View​

แสดงรายละเอียด Booking แบบ read-only

  • กดปุ่ม Edit เพื่อเข้าสู่โหมดแก้ไข

Edit​

แก้ไขข้อมูล Booking ที่มีสถานะ Reserved หรือ Confirmed

  1. แก้ไขข้อมูลในส่วนที่ต้องการ
  2. กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก
ข้อควรระวัง

Booking ที่มีสถานะ Completed จะไม่สามารถแก้ไขได้

Cancel​

ยกเลิก Booking — สถานะจะเปลี่ยนเป็น Canceled

ข้อควรระวัง

การยกเลิก Booking ไม่สามารถกู้คืนได้ ควรตรวจสอบให้แน่ใจก่อนดำเนินการ
