Create Booking

หน้า Create Booking ใช้สำหรับสร้างรายการ Booking ใหม่สำหรับลูกค้า โดยมีขั้นตอนการกรอกข้อมูลหลายส่วน

ขั้นตอนการสร้าง Booking​

1. เลือกวันที่เดินทาง (Booking Travel Date)​

ที่มุมขวาบนของฟอร์ม ให้เลือก Booking Travel Date ซึ่งคือวันที่ลูกค้าจะออกเดินทาง

2. กรอกข้อมูล Booking Details​

ฟิลด์คำอธิบายจำเป็น
Sellerเลือก Seller หรือตัวแทนที่รับจอง✅
Booking No.กรอกเลขที่ Booking✅
Program Nameเลือกชื่อโปรแกรมทัวร์✅
Paymentเลือกประเภทการชำระเงิน—
Optionเลือก Option เพิ่มเติม (ถ้ามี)—
Trip Typeเลือกประเภท Trip✅
Trip roundเลือกรอบ Trip✅
Languageเลือกภาษาที่ต้องการ✅

3. กรอกข้อมูล Customer Details​

ฟิลด์คำอธิบายจำเป็น
Customer Nameชื่อลูกค้า✅
Nationalityสัญชาติของลูกค้า—
Emailอีเมลลูกค้า—
Phone numberเบอร์โทรลูกค้า—
Contact methodช่องทางการติดต่อ—

4. กรอกข้อมูล Unit Details​

ส่วน Unit Details แสดงรายการสินค้าหรือบริการที่รวมอยู่ใน Program

คอลัมน์คำอธิบาย
Unitชื่อรายการ
Priceราคาต่อหน่วย
Quantityจำนวน
KBราคารวม (KB)

ที่ด้านล่างจะแสดง:

  • Sub total — ราคารวมก่อนหักส่วนลด
  • Discount — ส่วนลด (ถ้ามี)
  • Total Pax — จำนวนผู้เดินทางรวม
  • Total KB — ยอดรวม KB
  • Total Price — ราคารวมสุทธิ (บาท)

5. กรอกข้อมูล Location Details​

ฟิลด์คำอธิบาย
Drop-off pointจุด Drop-off ที่ต้องการ
Drop-off remarkหมายเหตุเพิ่มเติมเกี่ยวกับ Drop-off

6. กรอก Remark​

กรอกข้อความพิเศษหรือหมายเหตุเพิ่มเติมในช่อง Remark เช่น ความต้องการพิเศษของลูกค้า

7. ยืนยันการสร้าง Booking​

  • กดปุ่ม Create Booking เพื่อบันทึกข้อมูล
  • กดปุ่ม Clear Form เพื่อล้างข้อมูลทั้งหมดและเริ่มใหม่
ข้อควรระวัง

ฟิลด์ที่มีเครื่องหมาย * (ดอกจัน) เป็นฟิลด์บังคับ หากไม่กรอกจะไม่สามารถบันทึกข้อมูลได้

เคล็ดลับ

ควรสร้าง Seller, Program และ Trip round ใน Setting ก่อน เพื่อให้มีข้อมูลให้เลือกในฟอร์ม Create Booking
