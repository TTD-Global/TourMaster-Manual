Roles

หน้า Roles ใช้จัดการสิทธิ์การเข้าถึงระบบ สามารถสร้าง Role ใหม่และกำหนดว่า Role นั้นสามารถเข้าถึง Function ใดได้บ้าง

เข้าถึงได้จาก Setting → User → Roles

ตารางรายการ​

คอลัมน์คำอธิบาย
#ลำดับที่
Roles Listชื่อ Role
Statusสถานะการใช้งาน
Active เปิดใช้งาน
Inactive ปิดใช้งาน
Last Modify Dateวันที่สร้าง/แก้ไขล่าสุด
Actionปุ่ม ... สำหรับจัดการ

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Create Rolesสร้าง Role ใหม่
Searchค้นหา Role
Filterกรองข้อมูลตามเงื่อนไข

ตัวกรอง​

กดปุ่ม Filter เพื่อเปิด Modal กรองข้อมูลตามเงื่อนไขที่ต้องการ

ฟิลด์คำอธิบาย
Statusกรองตามสถานะ Active / Inactive

กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

กดปุ่ม ... ในคอลัมน์ Action เพื่อจัดการ Role

View​

แสดงรายละเอียด Role แบบ read-only

  • Role Detail — ชื่อ Role และ Status
  • Role Permissions — ตารางสิทธิ์ทั้งหมดที่กำหนดไว้ แบ่งกลุ่มตาม Function
  • กดปุ่ม Edit เพื่อเข้าสู่โหมดแก้ไข
  • กดปุ่ม Close เพื่อปิดและกลับไปหน้ารายการ

Edit​

กดปุ่ม Edit จากหน้า View เพื่อแก้ไขข้อมูล Role

ฟิลด์คำอธิบายจำเป็น
Role Nameชื่อ Role✅
Statusเลือก Active หรือ Inactive—
Role Permissionsกำหนดสิทธิ์แต่ละ Function (Select All / Unselect All)—

กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก

Create​

กดปุ่ม Create Roles มุมขวาบน ระบบจะนำไปยังฟอร์มสร้าง Role ใหม่ แบ่งเป็น 2 ส่วน

Role Detail​

ฟิลด์คำอธิบายจำเป็น
Role Nameตั้งชื่อสิทธิ์การเข้าถึง✅
Statusกำหนดสถานะ Active หรือ Inactive—

Role Permissions​

กำหนดสิทธิ์การใช้งานแต่ละ Function

  • Select All / Unselect All — เลือกหรือยกเลิกสิทธิ์ทั้งหมดพร้อมกัน
  • เลือก checkbox แต่ละ Function ที่ต้องการให้ Role นี้เข้าถึงได้

กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก
