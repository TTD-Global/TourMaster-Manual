User List

หน้า User List ใช้จัดการบัญชีผู้ใช้งานในระบบ ทั้งการสร้าง แก้ไขข้อมูล และกำหนดสิทธิ์การเข้าถึง

เข้าถึงได้จาก Setting → User → User List

ตารางรายการ​

คอลัมน์คำอธิบาย
#ลำดับที่
Imageรูปภาพโปรไฟล์
Employee Codeรหัสพนักงาน
Roleสิทธิ์การเข้าถึงที่กำหนดให้ผู้ใช้งาน
Usernameชื่อผู้ใช้สำหรับเข้าสู่ระบบ
Nameชื่อ-นามสกุล
Emailอีเมล
Last Loginวันและเวลาที่เข้าสู่ระบบล่าสุด
Actionปุ่ม ... สำหรับจัดการ

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Create Userสร้างบัญชีผู้ใช้งานใหม่
Searchค้นหาผู้ใช้งาน
Filterกรองข้อมูลตามเงื่อนไข

ตัวกรอง​

กดปุ่ม Filter เพื่อเปิด Modal กรองข้อมูลตามเงื่อนไขที่ต้องการ

ฟิลด์คำอธิบาย
Roleกรองตาม Role ที่กำหนดให้ User
Employee codeกรองตามรหัสพนักงาน
Usernameกรองตาม Username

กด Search เพื่อค้นหา หรือ Clear all เพื่อล้างเงื่อนไข

Action​

กดปุ่ม ... ในคอลัมน์ Action เพื่อจัดการผู้ใช้งาน

View​

แสดงข้อมูล General Detail และ Security ของผู้ใช้งานแบบ read-only

  • กดปุ่ม Edit เพื่อเข้าสู่โหมดแก้ไข
  • กดปุ่ม Close เพื่อปิดและกลับไปหน้ารายการ

Edit​

กดปุ่ม Edit จากหน้า View เพื่อแก้ไขข้อมูลผู้ใช้งาน

ฟิลด์คำอธิบาย
Statusเปลี่ยนสถานะ Active หรือ Inactive
Reset Passwordกดเพื่อเปลี่ยนรหัสผ่าน

กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก

Reset Password​

ฟิลด์คำอธิบายจำเป็น
New Passwordรหัสผ่านใหม่✅
Confirm Passwordยืนยันรหัสผ่านใหม่ให้ตรงกัน✅

กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก

Create​

กดปุ่ม Create User มุมขวาบน ระบบจะนำไปยังฟอร์มสร้างผู้ใช้งาน แบ่งเป็น 2 ส่วน

General Detail​

ฟิลด์คำอธิบายจำเป็น
Imageรูปภาพโปรไฟล์—
Employee Codeรหัสพนักงาน—
First Nameชื่อ—
Last Nameนามสกุล—
Emailอีเมล—
Phoneเบอร์โทรศัพท์—

Security​

ฟิลด์คำอธิบายจำเป็น
Roleสิทธิ์การเข้าถึงระบบ✅
Usernameชื่อหรือ ID สำหรับเข้าสู่ระบบ✅
Passwordรหัสผ่าน✅
Confirm Passwordยืนยันรหัสผ่านให้ตรงกับ Password✅

กด Save เพื่อบันทึก หรือ Cancel เพื่อยกเลิก
