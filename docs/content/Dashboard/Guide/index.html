Guide

Guide Dashboard คือ Dashboard ติดตามสถิติการทำงานของ Guide และ Staff แสดงจำนวน Guide ที่ใช้งาน, Trip ที่รับผิดชอบ และ Ranking การมอบหมายงาน

เข้าถึงได้จาก Dashboard → Guide

ภาพรวม​

ตัวกรองช่วงเวลา​

แท็บคำอธิบาย
Todayข้อมูลวันนี้
7 Daysข้อมูล 7 วันย้อนหลัง
30 Daysข้อมูล 30 วันย้อนหลัง
12 Monthsข้อมูล 12 เดือนย้อนหลัง

Summary Cards​

Cardคำอธิบาย
Total Guideจำนวน Guide ทั้งหมดในระบบ (แยก Guide / Staff)
Guide & Staff usedจำนวน Guide และ Staff ที่ถูกมอบหมายงานในช่วงนั้น (แยก Guide / Staff) พร้อม %
Total Tripจำนวน Trip ทั้งหมด (แยก Private Trip / Join In Trip) พร้อม %
Total Participantsจำนวนผู้เดินทางรวม พร้อม %

กราฟและตาราง​

ส่วนคำอธิบาย
Guide usage by programกราฟแสดงการใช้งาน Guide แยกตามโปรแกรม กดปุ่ม View all programs เพื่อดูทั้งหมด
Top guides assigned most tasksอันดับ Guide ที่ได้รับมอบหมายงานมากที่สุด กดปุ่ม View guide ranking เพื่อดูรายละเอียด
