Balance

Balance คือหน้าสรุปยอดเงินคงเหลือของแต่ละ Trip เปรียบเทียบระหว่างเงิน Advance ที่เบิกไปกับค่าใช้จ่ายจริงที่เกิดขึ้น

เข้าถึงได้จาก Payment → Balance

ตารางรายการ​

Summary Card​

ด้านบนแสดง Card สรุปภาพรวม

Cardคำอธิบาย
Amount Tripจำนวน Trip ทั้งหมด
Total Advanceยอดเงิน Advance รวม
Total Actualยอดค่าใช้จ่ายจริงรวม
Return to companyยอดเงินที่ต้องคืนบริษัท (Advance > Actual)
Company pays additionalยอดที่บริษัทต้องจ่ายเพิ่ม (Actual > Advance)

แท็บสถานะ​

แท็บคำอธิบาย
PendingTrip ที่ยังรอดำเนินการ
CompletedTrip ที่ดำเนินการเสร็จแล้ว

คอลัมน์ตาราง​

คอลัมน์คำอธิบาย
#ลำดับที่
Trip Codeรหัส Trip
Trip Typeประเภท Trip (เช่น Join In)
Programชื่อโปรแกรม
Guideชื่อ Guide ที่รับผิดชอบ
Adv.Paxจำนวน Pax ที่ใช้คำนวณ Advance
Adv.Costยอดเงิน Advance
Act.Paxจำนวน Pax ที่เกิดขึ้นจริง
Expenseยอดค่าใช้จ่ายจริง
Balanceยอดคงเหลือ (Adv.Cost − Expense)
Slipหลักฐานการโอนเงิน / สลิป
Actionปุ่มจัดการ

การตีความ Balance​

ผลลัพธ์ความหมาย
Balance > 0 (บวก)Advance มากกว่า Expense → Guide คืนเงินส่วนเกิน
Balance = 0ค่าใช้จ่ายตรงกับ Advance พอดี
Balance < 0 (ลบ)Expense เกิน Advance → บริษัทจ่ายเพิ่มให้ Guide

ฟังก์ชันบนหน้า​

ปุ่ม / ฟังก์ชันคำอธิบาย
Date Filterกรองรายการตามวันที่
Exportส่งออกรายงาน Balance เป็นไฟล์
info

Balance คำนวณอัตโนมัติจากข้อมูล Advance และ Expenses ที่บันทึกไว้ สามารถดูสรุปภาพรวมทุก Trip ได้ใน Report
